# M01 — 项目导览与环境搭建

> **前置知识**：P01（现代 CMake 与构建工具链）、P02（vcpkg 包管理）
> **难度**：★☆☆（入门级）

本模块带你从零认识 KrKr2 项目：了解 KiriKiri 引擎的历史渊源，掌握项目的代码架构与构建系统，在四大平台（Windows / Linux / macOS / Android）上搭建开发环境，并完成第一次编译和运行。

## 学习建议

1. **按顺序阅读**：第 1-2 章建立认知，第 3 章动手搭环境，第 4 章编译运行验证
2. **选择你的平台**：环境搭建章节可以先看你的主力开发平台，其他平台作为参考
3. **动手做练习**：每节末尾的练习题帮助巩固知识点
4. **遇到问题先查 4.3**：常见问题排查涵盖了绝大部分新手会遇到的坑

完成本模块后，你应该能够：
- 理解 KiriKiri 引擎的历史和 KrKr2 项目的技术选型
- 熟悉项目的代码组织和构建系统
- 在至少一个平台上成功编译并运行 KrKr2 模拟器
- 具备进入 M02（构建系统深度解析）的基础
