## 1. 图像加载器架构总览

KrKr2 的图像加载系统采用**回调驱动**（Callback-Driven）的插件式架构，核心设计思想是：

- **解耦格式与核心**：图像格式的解码逻辑与引擎核心完全分离
- **统一回调接口**：所有格式加载器通过相同的回调函数与引擎通信
- **扩展性优先**：支持运行时动态注册新格式，无需重新编译引擎

## 2. 回调函数类型详解

图像加载器与引擎核心通过 4 种回调函数通信：

### 2.1 tTVPGraphicSizeCallback — 尺寸通知回调

```cpp
// GraphicsLoaderIntf.h 第 28-33 行
typedef int(*tTVPGraphicSizeCallback)  // 返回行 pitch（字节数）
    (void *callbackdata,               // 引擎传入的上下文数据
     tjs_uint w,                       // 图像宽度（像素）
     tjs_uint h,                       // 图像高度（像素）
     tTVPGraphicPixelFormat fmt);      // 像素格式
```

**调用时机**：加载器解析图像头后，**必须首先调用此回调**通知引擎分配内存。

**像素格式枚举**：

```cpp
// GraphicsLoaderIntf.h 第 22 行
enum tTVPGraphicPixelFormat {
    gpfLuminance,   // 灰度图 (8-bit)
    gpfPalette,     // 调色板图 (8-bit 索引)
    gpfRGB,         // RGB (24-bit)
    gpfRGBA         // RGBA (32-bit)
};
```

**返回值**：行 pitch（每行字节数），用于加载器对齐扫描线数据。

### 2.2 tTVPGraphicScanLineCallback — 扫描线回调

```cpp
// GraphicsLoaderIntf.h 第 35-44 行
typedef void *(*tTVPGraphicScanLineCallback)(
    void *callbackdata,  // 引擎上下文
    tjs_int y);          // 行号 (0-based) 或 -1
```

**调用模式**（每行解码时）：

1. 调用 `scanlinecallback(data, y)` 获取第 y 行的写入缓冲区指针
2. 将解码后的像素数据写入该缓冲区
3. 调用 `scanlinecallback(data, -1)` 通知引擎该行写入完成

**特殊行为**：
- `y >= 0`：返回第 y 行的可写缓冲区指针
- `y == -1`：通知引擎上一行数据已写入完成，**必须返回 nullptr**
- 返回 `nullptr`（非 y=-1 时）：表示引擎请求停止加载

### 2.3 tTVPMetaInfoPushCallback — 元数据回调

```cpp
// GraphicsLoaderIntf.h 第 49-54 行
typedef void (*tTVPMetaInfoPushCallback)(
    void *callbackdata,    // 引擎上下文
    const ttstr &name,     // 元数据键名 (如 "Author", "Comment")
    const ttstr &value);   // 元数据值
```

**用途**：推送图像的元数据（EXIF、PNG tEXt、TLG tags 等）到引擎。

**常见元数据键**：

| 键名 | 来源格式 | 说明 |
|------|----------|------|
| `"Author"` | PNG tEXt | 作者信息 |
| `"Comment"` | JPEG COM | 图像注释 |
| `"Software"` | PNG tEXt | 创建软件 |
| `"LoopCount"` | GIF | 动画循环次数 |

### 2.4 tTVPGraphicLoadingHandler — 主加载函数

```cpp
// GraphicsLoaderIntf.h 第 63-67 行
typedef void (*tTVPGraphicLoadingHandler)(
    void *formatdata,                              // 格式专用数据
    void *callbackdata,                            // 引擎上下文
    tTVPGraphicSizeCallback sizecallback,          // 尺寸回调
    tTVPGraphicScanLineCallback scanlinecallback,  // 扫描线回调
    tTVPMetaInfoPushCallback metainfopushcallback, // 元数据回调
    tTJSBinaryStream *src,                         // 输入流
    tjs_int32 keyidx,                              // 色键索引
    tTVPGraphicLoadMode mode);                     // 加载模式
```

**加载模式枚举**：

```cpp
// GraphicsLoaderIntf.h 第 56-60 行
enum tTVPGraphicLoadMode {
    glmNormal,      // 正常模式：输出 32-bit ARGB
    glmPalettized,  // 调色板模式：保留 8-bit 索引
    glmGrayscale    // 灰度模式：输出 8-bit 灰度
};
```

## 3. tTVPGraphicHandlerType 处理器类型

每个图像格式需要注册一个 `tTVPGraphicHandlerType` 结构体：

```cpp
// GraphicsLoaderIntf.h 第 371-452 行
struct tTVPGraphicHandlerType {
    bool IsPlugin;              // 是否为插件版本（使用 IStream）
    ttstr Extension;            // 文件扩展名（小写，带点）
    
    // 加载处理器（二选一）
    union {
        tTVPGraphicLoadingHandler LoadHandler;           // 内部版
        tTVPGraphicLoadingHandlerForPlugin LoadHandlerPlugin;  // 插件版
    };
    
    // 头信息处理器（二选一）
    union {
        tTVPGraphicHeaderLoadingHandler HeaderHandler;
        tTVPGraphicHeaderLoadingHandlerForPlugin HeaderHandlerPlugin;
    };
    
    // 保存处理器（二选一）
    union {
        tTVPGraphicSaveHandler SaveHandler;
        tTVPGraphicSaveHandlerForPlugin SaveHandlerPlugin;
    };
    
    tTVPGraphicAcceptSaveHandler AcceptHandler;  // 保存格式检查
    void *FormatData;           // 格式专用数据指针
    
    // 便捷方法
    void Load(...);             // 调用加载处理器
    void Save(...);             // 调用保存处理器
    void Header(...);           // 调用头信息处理器
    bool AcceptSave(...);       // 检查是否支持保存
};
```

### 3.1 内部版 vs 插件版

KrKr2 支持两套回调接口：

| 版本 | 流类型 | 使用场景 |
|------|--------|----------|
| 内部版 | `tTJSBinaryStream*` | 引擎内部加载器 |
| 插件版 | `IStream*` | 外部 DLL 插件 |

**差异原因**：外部插件可能用不同编译器编译，使用 COM 的 `IStream` 接口保证 ABI 兼容。

## 4. tTVPGraphicType 单例与注册机制

### 4.1 单例结构

```cpp
// GraphicsLoaderIntf.cpp 第 160-248 行
class tTVPGraphicType {
public:
    tTJSHashTable<ttstr, tTVPGraphicHandlerType> Hash;  // 扩展名快速查找
    std::vector<tTVPGraphicHandlerType> Handlers;       // 所有处理器列表
    static bool Avail;  // 单例是否可用

    tTVPGraphicType() {
        // 注册内置格式（按优先级排序）
        Handlers.push_back(tTVPGraphicHandlerType(
            TJS_W(".pvr"), TVPLoadGraphicRouter, TVPLoadHeaderRouter, ...));
        Handlers.push_back(tTVPGraphicHandlerType(
            TJS_W(".jxr"), TVPLoadGraphicRouter, TVPLoadHeaderRouter, ...));
        // ... .bpg, .webp, .bmp, .jpeg, .jpg, .png, .tlg, .tlg5, .tlg6
        ReCreateHash();
        Avail = true;
    }
    
    void Register(const tTVPGraphicHandlerType &handler);
    void Unregister(const tTVPGraphicHandlerType &handler);
    void ReCreateHash();  // 重建哈希表
} static TVPGraphicType;  // 全局单例
```

## 5. Magic Number 路由机制

当文件扩展名缺失或无法识别时，引擎使用 Magic Number 自动检测格式：

### 5.1 路由器实现

```cpp
// GraphicsLoaderIntf.cpp 第 50-91 行
static void TVPLoadGraphicRouter(
    void *formatdata, void *callbackdata,
    tTVPGraphicSizeCallback sizecallback,
    tTVPGraphicScanLineCallback scanlinecallback,
    tTVPMetaInfoPushCallback metainfopushcallback,
    tTJSBinaryStream *src, tjs_int keyidx,
    tTVPGraphicLoadMode mode) 
{
    uint8_t header[16];
    tjs_uint64 origSrcPos = src->GetPosition();  // 保存原始位置
    
    if(src->Read(header, sizeof(header)) == sizeof(header)) {
        src->SetPosition(origSrcPos);  // 恢复位置
        
        // Magic Number 检测（按检测顺序）
        if(!memcmp(header, "BM", 2))
            return TVPLoadBMP(...);
            
        if(!memcmp(header, "\x89PNG", 4))
            return TVPLoadPNG(...);
            
        if(!memcmp(header, "TLG", 3))
            return TVPLoadTLG(...);
            
        if(!memcmp(header, "\xFF\xD8\xFF", 3) && 
           header[3] >= 0xE0 && header[3] <= 0xEF)
            return TVPLoadJPEG(...);
            
        if(!memcmp(header, "BPG", 3))
            return TVPLoadBPG(...);
            
        if(!memcmp(header, "RIFF", 4) && 
           !memcmp(header + 8, "WEBPVP8", 7))
            return TVPLoadWEBP(...);
            
        if(!memcmp(header, "\x49\x49\xbc\x01", 4))  // JPEG XR
            return TVPLoadJXR(...);
            
        if(!memcmp(header, "PVR\3", 4))
            return TVPLoadPVRv3(...);
    }
    
    TVPThrowExceptionMessage(TVPImageLoadError, TJS_W("Invalid image"));
}
```

### 5.2 支持的 Magic Number

| 格式 | Magic Bytes | 偏移 | 说明 |
|------|-------------|------|------|
| BMP | `"BM"` | 0 | BM 标识 |
| PNG | `\x89PNG` | 0 | PNG 签名 |
| TLG | `"TLG"` | 0 | KiriKiri 原生 |
| JPEG | `\xFF\xD8\xFF\xE0-EF` | 0 | SOI + APPn |
| BPG | `"BPG"` | 0 | BPG 签名 |
| WebP | `"RIFF" + "WEBPVP8"` | 0,8 | RIFF 容器 |
| JPEG XR | `\x49\x49\xBC\x01` | 0 | 小端标识 |
| PVRv3 | `"PVR\x03"` | 0 | PVR 版本 3 |

## 6. 动手实践：实现自定义图像加载器

### 6.1 需求：实现 TGA 格式加载器

TGA（Truevision Graphics Adapter）是一种简单的图像格式，非常适合学习：

```
TGA 文件结构：
┌────────────────────┐
│ Header (18 bytes)  │
├────────────────────┤
│ ID 字段 (可选)     │
├────────────────────┤
│ 调色板 (可选)      │
├────────────────────┤
│ 图像数据           │
└────────────────────┘
```

### 6.2 完整实现

```cpp
// LoadTGA.cpp - TGA 格式加载器完整实现

#include "GraphicsLoaderIntf.h"
#include "MsgIntf.h"
#include "tvpgl.h"
#include <cstring>

//--------------------------------------------------------------
// TGA 文件头结构
//--------------------------------------------------------------
#pragma pack(push, 1)
struct TGAHeader {
    uint8_t  idLength;        // ID 字段长度
    uint8_t  colorMapType;    // 0=无调色板, 1=有调色板
    uint8_t  imageType;       // 图像类型
    // 调色板规格
    uint16_t colorMapOrigin;  // 调色板起始索引
    uint16_t colorMapLength;  // 调色板条目数
    uint8_t  colorMapDepth;   // 每条目位数
    // 图像规格
    uint16_t xOrigin;         // X 起点
    uint16_t yOrigin;         // Y 起点
    uint16_t width;           // 宽度
    uint16_t height;          // 高度
    uint8_t  bitsPerPixel;    // 每像素位数
    uint8_t  imageDescriptor; // 图像描述符
};
#pragma pack(pop)

// TGA 图像类型
enum TGAImageType {
    TGA_NO_IMAGE = 0,
    TGA_INDEXED = 1,
    TGA_RGB = 2,
    TGA_GRAYSCALE = 3,
    TGA_RLE_INDEXED = 9,
    TGA_RLE_RGB = 10,
    TGA_RLE_GRAYSCALE = 11
};

//--------------------------------------------------------------
// RLE 解码辅助函数
//--------------------------------------------------------------
static void DecodeTGARLE(
    tTJSBinaryStream *src,
    uint8_t *dest,
    int pixelCount,
    int bytesPerPixel)
{
    int pixelsDecoded = 0;
    
    while (pixelsDecoded < pixelCount) {
        uint8_t header;
        src->Read(&header, 1);
        
        int count = (header & 0x7F) + 1;  // 像素计数
        
        if (header & 0x80) {
            // RLE 包：重复单个像素
            uint8_t pixel[4];
            src->Read(pixel, bytesPerPixel);
            
            for (int i = 0; i < count; i++) {
                memcpy(dest, pixel, bytesPerPixel);
                dest += bytesPerPixel;
            }
        } else {
            // 原始包：直接读取像素
            src->Read(dest, count * bytesPerPixel);
            dest += count * bytesPerPixel;
        }
        
        pixelsDecoded += count;
    }
}

//--------------------------------------------------------------
// 主加载函数
//--------------------------------------------------------------
void TVPLoadTGA(
    void *formatdata,
    void *callbackdata,
    tTVPGraphicSizeCallback sizecallback,
    tTVPGraphicScanLineCallback scanlinecallback,
    tTVPMetaInfoPushCallback metainfopushcallback,
    tTJSBinaryStream *src,
    tjs_int keyidx,
    tTVPGraphicLoadMode mode)
{
    // 1. 读取文件头
    TGAHeader header;
    if (src->Read(&header, sizeof(header)) != sizeof(header)) {
        TVPThrowExceptionMessage(
            TVPImageLoadError, TJS_W("TGA header read failed"));
    }
    
    // 2. 验证图像类型
    bool isRLE = (header.imageType & 0x08) != 0;
    int baseType = header.imageType & 0x07;
    
    if (baseType != TGA_RGB && baseType != TGA_GRAYSCALE) {
        TVPThrowExceptionMessage(
            TVPImageLoadError, TJS_W("Unsupported TGA image type"));
    }
    
    // 3. 跳过 ID 字段
    if (header.idLength > 0) {
        src->SetPosition(src->GetPosition() + header.idLength);
    }
    
    // 4. 确定像素格式
    tTVPGraphicPixelFormat fmt;
    int bytesPerPixel;
    
    if (baseType == TGA_GRAYSCALE) {
        fmt = gpfLuminance;
        bytesPerPixel = 1;
    } else {
        switch (header.bitsPerPixel) {
            case 24:
                fmt = gpfRGB;
                bytesPerPixel = 3;
                break;
            case 32:
                fmt = gpfRGBA;
                bytesPerPixel = 4;
                break;
            default:
                TVPThrowExceptionMessage(
                    TVPImageLoadError, TJS_W("Unsupported TGA bit depth"));
        }
    }
    
    // 5. 通知引擎图像尺寸（必须先调用！）
    int pitch = sizecallback(
        callbackdata, 
        header.width, 
        header.height, 
        fmt);
    
    if (pitch == 0) {
        return;  // 引擎请求取消加载
    }
    
    // 6. 分配行缓冲区
    std::vector<uint8_t> rowBuffer(header.width * bytesPerPixel);
    
    // 7. 判断行顺序（TGA 可以是上下颠倒的）
    bool topToBottom = (header.imageDescriptor & 0x20) != 0;
    
    // 8. 逐行解码
    for (int i = 0; i < header.height; i++) {
        // 计算实际行号
        int y = topToBottom ? i : (header.height - 1 - i);
        
        // 读取一行数据
        if (isRLE) {
            DecodeTGARLE(
                src, 
                rowBuffer.data(), 
                header.width, 
                bytesPerPixel);
        } else {
            src->Read(rowBuffer.data(), header.width * bytesPerPixel);
        }
        
        // 获取目标缓冲区
        void *destLine = scanlinecallback(callbackdata, y);
        if (destLine == nullptr) {
            return;  // 引擎请求停止
        }
        
        // 转换像素格式（TGA 是 BGR/BGRA，需要转为 RGB/RGBA）
        if (fmt == gpfRGB || fmt == gpfRGBA) {
            uint8_t *src_ptr = rowBuffer.data();
            uint8_t *dst_ptr = (uint8_t*)destLine;
            
            for (int x = 0; x < header.width; x++) {
                dst_ptr[0] = src_ptr[2];  // R <- B
                dst_ptr[1] = src_ptr[1];  // G <- G
                dst_ptr[2] = src_ptr[0];  // B <- R
                
                if (bytesPerPixel == 4) {
                    dst_ptr[3] = src_ptr[3];  // A <- A
                }
                
                src_ptr += bytesPerPixel;
                dst_ptr += bytesPerPixel;
            }
        } else {
            // 灰度图直接复制
            memcpy(destLine, rowBuffer.data(), header.width);
        }
        
        // 通知行写入完成
        scanlinecallback(callbackdata, -1);
    }
    
    // 9. 推送元数据（可选）
    if (metainfopushcallback) {
        ttstr typeStr;
        switch (baseType) {
            case TGA_RGB: 
                typeStr = isRLE ? TJS_W("RGB RLE") : TJS_W("RGB"); 
                break;
            case TGA_GRAYSCALE: 
                typeStr = isRLE ? TJS_W("Grayscale RLE") : TJS_W("Grayscale"); 
                break;
        }
        metainfopushcallback(callbackdata, TJS_W("ImageType"), typeStr);
        
        ttstr bppStr = ttstr(header.bitsPerPixel);
        metainfopushcallback(callbackdata, TJS_W("BitsPerPixel"), bppStr);
    }
}

//--------------------------------------------------------------
// 头信息加载函数
//--------------------------------------------------------------
void TVPLoadHeaderTGA(
    void *formatdata,
    tTJSBinaryStream *src,
    iTJSDispatch2 **dic)
{
    TGAHeader header;
    if (src->Read(&header, sizeof(header)) != sizeof(header)) {
        TVPThrowExceptionMessage(
            TVPImageLoadError, TJS_W("TGA header read failed"));
    }
    
    // 创建字典对象
    *dic = TJSCreateDictionaryObject();
    
    tTJSVariant val;
    
    // width
    val = (tjs_int)header.width;
    (*dic)->PropSet(TJS_MEMBERENSURE, TJS_W("width"), nullptr, &val, *dic);
    
    // height
    val = (tjs_int)header.height;
    (*dic)->PropSet(TJS_MEMBERENSURE, TJS_W("height"), nullptr, &val, *dic);
    
    // bpp
    val = (tjs_int)header.bitsPerPixel;
    (*dic)->PropSet(TJS_MEMBERENSURE, TJS_W("bpp"), nullptr, &val, *dic);
    
    // grayscale
    bool isGray = (header.imageType & 0x07) == TGA_GRAYSCALE;
    val = isGray;
    (*dic)->PropSet(TJS_MEMBERENSURE, TJS_W("grayscale"), nullptr, &val, *dic);
}

//--------------------------------------------------------------
// 注册函数（引擎启动时调用）
//--------------------------------------------------------------
static struct TGALoaderRegistrar {
    TGALoaderRegistrar() {
        TVPRegisterGraphicLoadingHandler(
            TJS_W(".tga"),
            TVPLoadTGA,
            TVPLoadHeaderTGA,
            nullptr,  // 暂不支持保存
            nullptr,
            nullptr);
    }
    
    ~TGALoaderRegistrar() {
        TVPUnregisterGraphicLoadingHandler(
            TJS_W(".tga"),
            TVPLoadTGA,
            TVPLoadHeaderTGA,
            nullptr,
            nullptr,
            nullptr);
    }
} static_tga_registrar;  // 静态实例自动注册
```

### 6.3 代码要点说明

1. **静态注册模式**：使用静态变量的构造/析构函数自动注册/注销
2. **回调顺序**：`sizecallback` → `scanlinecallback(y)` → 写入 → `scanlinecallback(-1)`
3. **BGR 转 RGB**：TGA 使用 BGR 顺序，需要交换红蓝通道
4. **行顺序处理**：TGA 支持上下颠倒存储，需检查 `imageDescriptor`

## 8. 常见错误与排查

### 8.1 错误：忘记调用 sizecallback

**症状**：引擎崩溃或 `scanlinecallback` 返回 nullptr

**修复**：确保在任何 `scanlinecallback` 调用前先调用 `sizecallback`

### 8.2 错误：忘记调用 scanlinecallback(-1)

**症状**：图像显示不完整或有条纹

**修复**：每行写入后必须调用 `scanlinecallback(callbackdata, -1)`

### 8.3 错误：扩展名大小写问题

**症状**：`.TGA` 文件无法识别

**修复**：注册时使用小写扩展名，引擎内部会自动转换

```cpp
// 正确
TVPRegisterGraphicLoadingHandler(TJS_W(".tga"), ...);

// 错误
TVPRegisterGraphicLoadingHandler(TJS_W(".TGA"), ...);  // 无法匹配
```

### 8.4 错误：流位置未恢复

**症状**：Magic Number 检测后实际加载失败

**修复**：在 Router 中检测 Magic Number 后必须恢复流位置

```cpp
tjs_uint64 origPos = src->GetPosition();
src->Read(header, 16);
src->SetPosition(origPos);  // 恢复！
```

### 题目 1：回调顺序验证

以下代码有什么问题？如何修复？

```cpp
void MyLoader(...) {
    // 读取头信息
    int w = 640, h = 480;
    
    // 直接开始写入像素
    for (int y = 0; y < h; y++) {
        void *line = scanlinecallback(callbackdata, y);
        // ... 写入像素 ...
```

<details>
<summary>查看答案</summary>

**问题**：缺少 `sizecallback` 调用，引擎不知道图像尺寸，无法分配内存。

**修复**：

```cpp
void MyLoader(...) {
    int w = 640, h = 480;
    
    // 必须先调用 sizecallback 通知引擎分配内存
    int pitch = sizecallback(callbackdata, w, h, gpfRGBA);
    if (pitch == 0) return;  // 引擎请求取消
    
    for (int y = 0; y < h; y++) {
        void *line = scanlinecallback(callbackdata, y);
        // ... 写入像素 ...
        scanlinecallback(callbackdata, -1);  // 通知写入完成
    }
}
```

### 题目 3：实现只读头信息的加载器

实现一个 ICO 格式的 `TVPLoadHeaderICO` 函数，只读取图标的尺寸信息。

ICO 文件结构：
- 偏移 0-1：保留（0）
- 偏移 2-3：类型（1=ICO, 2=CUR）
- 偏移 4-5：图标数量
- 每个图标条目 16 字节，偏移 0-1 是宽高（0 表示 256）

<details>
<summary>查看答案</summary>

```cpp
#pragma pack(push, 1)
struct ICOHeader {
    uint16_t reserved;     // 必须为 0
    uint16_t type;         // 1=ICO, 2=CUR
    uint16_t count;        // 图标数量
};

struct ICOEntry {
    uint8_t  width;        // 宽度（0=256）
    uint8_t  height;       // 高度（0=256）
    uint8_t  colorCount;   // 调色板颜色数
    uint8_t  reserved;     // 保留
    uint16_t planes;       // 颜色平面数
    uint16_t bitCount;     // 每像素位数
    uint32_t bytesInRes;   // 数据大小
    uint32_t imageOffset;  // 数据偏移
};
#pragma pack(pop)

void TVPLoadHeaderICO(
    void *formatdata,
    tTJSBinaryStream *src,
    iTJSDispatch2 **dic)
{
    // 读取 ICO 头
    ICOHeader header;
    if (src->Read(&header, sizeof(header)) != sizeof(header)) {
        TVPThrowExceptionMessage(
            TVPImageLoadError, TJS_W("ICO header read failed"));
    }
    
    // 验证格式
    if (header.reserved != 0 || (header.type != 1 && header.type != 2)) {
        TVPThrowExceptionMessage(
            TVPImageLoadError, TJS_W("Invalid ICO format"));
    }
    
    if (header.count == 0) {
        TVPThrowExceptionMessage(
            TVPImageLoadError, TJS_W("ICO contains no images"));
    }
    
    // 读取第一个图标条目
    ICOEntry entry;
    if (src->Read(&entry, sizeof(entry)) != sizeof(entry)) {
        TVPThrowExceptionMessage(
            TVPImageLoadError, TJS_W("ICO entry read failed"));
    }
    
    // 创建字典
    *dic = TJSCreateDictionaryObject();
    tTJSVariant val;
    
    // width（0 表示 256）
    int width = (entry.width == 0) ? 256 : entry.width;
    val = (tjs_int)width;
    (*dic)->PropSet(TJS_MEMBERENSURE, TJS_W("width"), nullptr, &val, *dic);
    
    // height（0 表示 256）
    int height = (entry.height == 0) ? 256 : entry.height;
    val = (tjs_int)height;
    (*dic)->PropSet(TJS_MEMBERENSURE, TJS_W("height"), nullptr, &val, *dic);
    
    // bpp
    val = (tjs_int)entry.bitCount;
    (*dic)->PropSet(TJS_MEMBERENSURE, TJS_W("bpp"), nullptr, &val, *dic);
    
    // count（图标数量）
    val = (tjs_int)header.count;
    (*dic)->PropSet(TJS_MEMBERENSURE, TJS_W("count"), nullptr, &val, *dic);
}
```

</details>
