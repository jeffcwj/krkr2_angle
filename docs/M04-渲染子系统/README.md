## 模块概述

本模块深入讲解 KrKr2 引擎的渲染子系统（`cpp/core/visual/`），这是项目中最复杂的核心模块之一。渲染子系统负责：

- **图层管理**：实现视觉小说的多层图像叠加、透明度、变换
- **图像加载**：支持 PNG/JPEG/WebP/BPG/JXR/TLG 等多种格式
- **GPU 渲染**：基于 OpenGL/OpenGL ES 的硬件加速渲染
- **软件渲染**：CPU 像素级操作、Alpha 混合、SIMD 优化
- **过渡效果**：场景切换时的淡入淡出、百叶窗、马赛克等特效

## 学习目标

完成本模块后，你将能够：

1. **理解渲染架构**：画出完整的渲染管线图，说清楚每个组件的职责
2. **分析图层系统**：理解 LayerIntf/LayerBitmapIntf/LayerManager 的设计模式
3. **添加图像格式**：按照 Load*.cpp 模式，添加新的图像格式支持
4. **修改 OGL 渲染**：理解纹理上传、批量渲染、移动端纹理压缩
5. **优化软件渲染**：掌握 tvpgl 像素库，理解 SSE/ASM 优化路径
6. **自定义过渡效果**：实现新的场景切换特效
