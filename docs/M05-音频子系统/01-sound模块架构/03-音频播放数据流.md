## 数据流总览

KrKr2 音频播放采用**生产者-消费者**模式，通过双层缓冲实现解码与播放的解耦：

```
┌─────────────────────────────────────────────────────────────────────────┐
│                        音频播放数据流架构                                  │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                          │
│  ┌──────────┐    ┌──────────────┐    ┌─────────────┐    ┌────────────┐ │
│  │ 音频文件  │───▶│ tTVPWave     │───▶│ Filter      │───▶│ L2 Buffer  │ │
│  │ OGG/WAV  │    │ Decoder      │    │ Chain       │    │ (解码缓存)  │ │
│  │ FLAC/... │    │              │    │             │    │            │ │
│  └──────────┘    └──────────────┘    └─────────────┘    └─────┬──────┘ │

│                        解码线程 (tTVPWaveSoundBufferDecodeThread)        │

```

## L1/L2 双层缓冲机制

### 为什么需要双层缓冲？

单层缓冲的问题：
1. **解码与播放耦合**：解码速度不稳定（压缩率不同、I/O 延迟），直接影响播放流畅性
2. **线程阻塞**：播放线程等待解码时会造成音频卡顿
3. **CPU 抖动**：解码消耗 CPU 不均匀，影响游戏帧率

双层缓冲的解决方案：

```
┌─────────────────────────────────────────────────────────────────┐
│                    L2 Buffer（解码缓存）                          │
│  ┌───┬───┬───┬───┬───┬───┬───┬───┬───┬───┬───┬───┬───┬───┬───┐ │
│  │ U │ U │ U │ U │ U │ U │ U │ U │ W │ R │ D │ D │ D │ D │ D │ │
│  │ 0 │ 1 │ 2 │ 3 │ 4 │ 5 │ 6 │ 7 │   │   │ 10│ 11│ 12│ 13│ 14│ │
│  └───┴───┴───┴───┴───┴───┴───┴───┴───┴───┴───┴───┴───┴───┴───┘ │
│                                    ↑   ↑                        │
│                         WritePos ──┘   └── ReadPos              │
│  U = 未使用, D = 已解码待读, W = 正在写入, R = 正在读取          │
│  默认 L2BufferUnits = TVPL2BufferLength / (1000/TVP_WSB_ACCESS_FREQ) │
│                      = 1000ms / (1000/50) = 50 个单元             │
└─────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────┐
│                    L1 Buffer（播放缓冲）                          │
│  ┌───────┬───────┬───────┬───────┬───────┬───────┬───────┬───┐ │
│  │ Unit0 │ Unit1 │ Unit2 │ Unit3 │ Unit4 │ Unit5 │ Unit6 │...│ │
│  │Playing│ Ready │ Ready │ Ready │Writing│ Empty │ Empty │   │ │
│  └───────┴───────┴───────┴───────┴───────┴───────┴───────┴───┘ │
│  默认 L1BufferUnits = TVPAL_BUFFER_COUNT = 4                    │
└─────────────────────────────────────────────────────────────────┘
```

```cpp
// WaveImpl.cpp:1997-2038 — TryCreateSoundBuffer() 中的缓冲区分配

// AccessUnitSamples: 每个缓冲单元的采样数
// TVP_WSB_ACCESS_FREQ 定义为 50（Hz），即每秒访问 50 次
AccessUnitSamples = InputFormat.SamplesPerSec / TVP_WSB_ACCESS_FREQ;
// 例如 44100Hz 音频：44100 / 50 = 882 samples/unit

// AccessUnitBytes: 每个缓冲单元的字节数
AccessUnitBytes = AccessUnitSamples * InputFormat.Channels * InputFormat.BytesPerSample;
// 例如 16bit 立体声：882 * 2 * 2 = 3528 bytes/unit

// L1 缓冲单元数（固定为 TVPAL_BUFFER_COUNT）
L1BufferUnits = TVPAL_BUFFER_COUNT; // 定义在 WaveMixer.h:17，值为 4

// L2 缓冲单元数
L2BufferUnits = TVPL2BufferLength / (1000 / TVP_WSB_ACCESS_FREQ);
// 1000ms / 20ms = 50 个单元

// 分配 L2 缓冲区
L2AccessUnitBytes = AccessUnitSamples * InputFormat.BytesPerSample * InputFormat.Channels;
Level2BufferSize = L2AccessUnitBytes * L2BufferUnits;
Level2Buffer = new tjs_uint8[Level2BufferSize];
// 例如：3528 * 50 = 176,400 bytes ≈ 172KB
```

### 解码线程主循环

```cpp
// WaveImpl.cpp:1828-1868 — 解码线程 Execute()

#define TVP_WSB_DECODE_THREAD_SLEEP_TIME 110  // 缓冲满时的睡眠时间（ms）

void tTVPWaveSoundBufferDecodeThread::Execute() {
    while(!GetTerminated()) {
        // 外层循环：等待启动信号
        DWORD st = TVPGetTickCount();
        
        while(Running) {
            // 内层循环：持续解码直到停止或缓冲满
            bool wait;
            DWORD et;

            if(Running) {
                // 加锁执行一次解码
                volatile tTJSCriticalSectionHolder cs_holder(OneLoopCS);
                wait = !Owner->FillL2Buffer(false, true); // 填充 L2 缓冲
                // 返回 false 表示缓冲已满，需要等待
            }

            if(GetTerminated()) break;

            if(Running) {
                et = TVPGetTickCount();
                TVPPushEnvironNoise(&et, sizeof(et)); // 收集随机熵
                
                if(wait) {
                    // 缓冲已满，睡眠等待
                    DWORD elapsed = et - st;
                    if(elapsed < TVP_WSB_DECODE_THREAD_SLEEP_TIME) {
                        Event.WaitFor(TVP_WSB_DECODE_THREAD_SLEEP_TIME - elapsed);
                    }
                } else {
                    // 缓冲未满，让出 CPU 但继续解码
                    TVPRelinquishCPU();
                    if(!GetTerminated())
                        SetPriority(TVPDecodeThreadLowPriority); // 降低优先级
                }
                st = et;
            }
        }
        
        if(GetTerminated()) break;
        
        // Running=false 时休眠等待
        Event.WaitFor(0);
    }
}
```

### FillL2Buffer 解码流程

```cpp
// WaveImpl.cpp:2395-2467 — 填充 L2 缓冲区

bool tTJSNI_WaveSoundBuffer::FillL2Buffer(bool firstwrite, bool fromdecodethread) {
    // 提升解码线程优先级（如果从主线程调用）
    if(!fromdecodethread && Thread->GetRunning())
        Thread->SetPriority(ttpHighest);

    tTJSCriticalSectionHolder holder(L2BufferCS);

    // 首次写入时重置所有指针
    if(firstwrite) {
        L2BufferReadPos = L2BufferWritePos = L2BufferRemain = 0;
        L2BufferEnded = false;
        for(tjs_int i = 0; i < L2BufferUnits; i++)
            L2BufferDecodedSamplesInUnit[i] = 0;
    }

    // 检查缓冲是否已满
    {
        tTJSCriticalSectionHolder holder(L2BufferRemainCS);
        if(L2BufferRemain == L2BufferUnits)
            return false; // 缓冲已满，返回 false
    }
    
    // 缓冲未满，更新滤波器链状态
    UpdateFilterChain();

    // 解码一个单元
    if(L2BufferEnded) {
        // 解码已结束，清空当前单元
        L2BufferSegmentQueues[L2BufferWritePos].Clear();
        L2BufferDecodedSamplesInUnit[L2BufferWritePos] = 0;
    } else {
        // 调用 Decode() 解码
        L2BufferSegmentQueues[L2BufferWritePos].Clear();
        tjs_uint decoded = Decode(
            L2BufferWritePos * L2AccessUnitBytes + Level2Buffer, // 目标地址
            AccessUnitSamples,                                   // 请求采样数
            L2BufferSegmentQueues[L2BufferWritePos]              // 分段队列
        );

        // 检查是否解码结束
        if(decoded < (tjs_uint)AccessUnitSamples)
            L2BufferEnded = true;

        L2BufferDecodedSamplesInUnit[L2BufferWritePos] = decoded;
    }

    // 移动写指针（环形缓冲）
    L2BufferWritePos++;
    if(L2BufferWritePos >= L2BufferUnits)
        L2BufferWritePos = 0;

    // 增加剩余计数
    {
        tTJSCriticalSectionHolder holder(L2BufferRemainCS);
        L2BufferRemain++;
    }

    return true; // 返回 true 表示还可以继续解码
}
```

```cpp
// WaveImpl.cpp:1583-1651 — 播放线程 Execute()

#define TVP_WSB_THREAD_SLEEP_TIME 60  // 基础睡眠时间（ms）

void tTVPWaveSoundBufferThread::Execute() {
    while(!GetTerminated()) {
        DWORD time = TVPGetRoughTickCount32();
        TVPPushEnvironNoise(&time, sizeof(time));

        { // 线程保护区块
            tTJSCriticalSectionHolder holder(TVPWaveSoundBufferVectorCS);

            // 处理主缓冲状态变化
            if(TVPPrimaryBufferPlayingByProgram != TVPPrimarySoundBufferPlaying) {
                TVPPrimarySoundBufferPlaying = TVPPrimaryBufferPlayingByProgram;
                // 通知所有 SoundBuffer 暂停/恢复
                for(auto i = TVPWaveSoundBufferVector.begin();
                    i != TVPWaveSoundBufferVector.end(); i++) {
                    if((*i)->ThreadCallbackEnabled)
                        (*i)->SetBufferPaused(!TVPPrimaryBufferPlayingByProgram);
                }
            }

            // 处理待触发的标签事件
            if(PendingLabelEventExists && !WndProcToBeCalled) {
                WndProcToBeCalled = true;
                EventQueue.PostEvent(NativeEvent(TVP_EV_WAVE_SND_BUF_THREAD));
            }

            // 定时填充所有活跃的 SoundBuffer
            if(TVPPrimarySoundBufferPlaying &&
               time - LastFilledTick >= TVP_WSB_THREAD_SLEEP_TIME) {
                for(auto i = TVPWaveSoundBufferVector.begin();
                    i != TVPWaveSoundBufferVector.end(); i++) {
                    if((*i)->ThreadCallbackEnabled)
                        (*i)->FillBuffer(); // 填充 L1 缓冲
                }
                LastFilledTick = time;
            }
        } // 线程保护区块结束

        // 计算睡眠时间
        DWORD time2 = TVPGetRoughTickCount32();
        time = time2 - time;

        if(time < TVP_WSB_THREAD_SLEEP_TIME) {
            tjs_int sleep_time = TVP_WSB_THREAD_SLEEP_TIME - time;
            if(PendingLabelEventExists) {
                tjs_int step_to_next = NextLabelEventTick - time2;
                if(step_to_next < sleep_time)
                    sleep_time = step_to_next;
                if(sleep_time < 1) sleep_time = 1;
            }
            Event.WaitFor(sleep_time);
        } else {
            Event.WaitFor(1);
        }
    }
}
```

### FillBuffer 填充流程

```cpp
// WaveImpl.cpp:2579-2766 — 填充 L1 缓冲区（关键代码摘录）

bool tTJSNI_WaveSoundBuffer::FillBuffer(bool firstwrite, bool allowpause) {
    tTJSCriticalSectionHolder holder(BufferCS);

    // 前置检查
    if(!SoundBuffer || !Decoder || !BufferPlaying || !TVPPrimarySoundBufferPlaying)
        return true;

    // 处理暂停状态
    if(allowpause && Paused) {
        if(DSBufferPlaying) {
            SoundBuffer->Pause();
            DSBufferPlaying = false;
        }
        return true;
    }

    // 检查 L2 缓冲剩余量
    tjs_int bufferremain;
    {
        tTJSCriticalSectionHolder holder(L2BufferRemainCS);
        bufferremain = L2BufferRemain;
    }

    // L2 缓冲不足 1 秒时提升解码线程优先级
    if(Thread->GetRunning() && bufferremain < TVP_WSB_ACCESS_FREQ)
        Thread->SetPriority(ttpNormal);

    // 首次写入
    if(firstwrite) {
        writepos = 0;
        PlayStopPos = -1;
        SoundBufferWritePos = 1;
        SoundBufferPrevReadPos = 0;
    } else {
        // 检查播放是否结束
        if(L2BufferEnded && SoundBuffer->GetRemainBuffers() == 0) {
            FlushAllLabelEvents();      // 触发所有剩余标签事件
            SoundBuffer->Pause();
            ResetSamplePositions();
            DSBufferPlaying = false;
            BufferPlaying = false;
            if(LoopManager) LoopManager->SetPosition(0);
            return true;
        }

        // 检查 OpenAL 缓冲区是否已满
        if(SoundBuffer->GetRemainBuffers() >= TVPAL_BUFFER_COUNT) {
            return true; // 缓冲已满，等待播放消耗
        }

        writepos = SoundBufferWritePos * AccessUnitBytes;
        SoundBufferWritePos++;
        if(SoundBufferWritePos >= L1BufferUnits)
            SoundBufferWritePos = 0;
    }

    // 从 L2 读取并写入 L1
    if(bufferremain > 1) {
        // L2 有足够数据，直接读取
        FillDSBuffer(writepos, *segment);
    } else {
        // L2 数据不足，同步等待解码
        PrepareToReadL2Buffer(false);
        {
            tTJSCriticalSectionHolder l2holder(L2BufferCS);
            FillDSBuffer(writepos, *segment);
        }
    }

    // 处理标签事件
    const std::deque<tTVPWaveLabel> &labels = segment->GetLabels();
    if(labels.size() != 0) {
        for(auto i = labels.begin(); i != labels.end(); i++) {
            LabelEventQueue.emplace_back(i->Position, i->Name, i->Offset + DecodePos);
        }
        std::sort(LabelEventQueue.begin(), LabelEventQueue.end(),
                  tTVPWaveLabel::tSortByOffsetFuncObj());
        TVPReschedulePendingLabelEvent(GetNearestEventStep());
    }

    DecodePos += AccessUnitSamples;
    return false;
}
```

### FillDSBuffer 和 ReadL2Buffer

```cpp
// WaveImpl.cpp:2528-2539 — 填充 DirectSound/OpenAL 缓冲

void tTJSNI_WaveSoundBuffer::FillDSBuffer(tjs_int writepos,
                                          tTVPWaveSegmentQueue &segments) {
    segments.Clear();
    if(SoundBuffer->IsBufferValid()) {
        tjs_uint decoded = ReadL2Buffer(
            UseVisBuffer ? VisBuffer + writepos : nullptr,
            segments
        );
    }
}

// WaveImpl.cpp:2481-2525 — 从 L2 缓冲读取数据

tjs_uint tTJSNI_WaveSoundBuffer::ReadL2Buffer(void *buffer,
                                              tTVPWaveSegmentQueue &segments) {
    // 获取当前单元的已解码采样数
    tjs_uint decoded = L2BufferDecodedSamplesInUnit[L2BufferReadPos];
    segments = L2BufferSegmentQueues[L2BufferReadPos];
    
    if(decoded) {
        // 将数据追加到 OpenAL 缓冲队列
        SoundBuffer->AppendBuffer(
            L2BufferReadPos * L2AccessUnitBytes + Level2Buffer,  // 源地址
            decoded * InputFormat.BytesPerSample * InputFormat.Channels // 字节数
        );
        
        // 如果需要可视化缓冲，复制数据
        if(buffer) {
            memcpy(buffer, 
                   L2BufferReadPos * L2AccessUnitBytes + Level2Buffer,
                   decoded * InputFormat.BytesPerSample * InputFormat.Channels);
        }
    }
    
    // 用静音填充不足的部分
    if(buffer && decoded < (tjs_uint)AccessUnitSamples) {
        TVPMakeSilentWave(
            (tjs_uint8 *)buffer + decoded * InputFormat.Channels * InputFormat.BytesPerSample,
            AccessUnitSamples - decoded,
            &InputFormat
        );
    }

    // 移动读指针（环形缓冲）
    L2BufferReadPos++;
    if(L2BufferReadPos >= L2BufferUnits)
        L2BufferReadPos = 0;

    // 减少剩余计数
    {
        tTJSCriticalSectionHolder holder(L2BufferRemainCS);
        L2BufferRemain--;
    }

    return decoded;
}
```

## 本节小结

- KrKr2 使用**双层缓冲**（L1/L2）解耦解码与播放，L2 作为解码缓存，L1 作为播放缓冲
- **解码线程**独立运行，持续填充 L2 缓冲，优先级根据缓冲状态动态调整
- **播放线程**定期从 L2 读取数据填充 L1，并驱动 OpenAL 播放
- 播放启动时预填充 4 个 L1 单元，确保开始播放时有足够数据
- 标签事件通过 `LabelEventQueue` 队列管理，按时间顺序触发

## 练习题与答案

### 题目 1：缓冲区大小计算

假设音频格式为 48000Hz、16bit、立体声，L2 缓冲长度设为 2000ms，计算：
1. 每个缓冲单元的采样数和字节数
2. L2 缓冲区的总单元数和总大小

<details>
<summary>查看答案</summary>

```
1. 每个缓冲单元的采样数和字节数：
   - AccessUnitSamples = 48000 / 50 = 960 samples
   - AccessUnitBytes = 960 * 2 (channels) * 2 (bytes) = 3840 bytes

2. L2 缓冲区的总单元数和总大小：
   - L2BufferUnits = 2000 / (1000/50) = 2000 / 20 = 100 个单元
   - Level2BufferSize = 3840 * 100 = 384,000 bytes ≈ 375KB
```

</details>

### 题目 2：线程同步问题

阅读以下代码，解释为什么需要两个不同的临界区锁（L2BufferCS 和 L2BufferRemainCS）：

```cpp
tTJSCriticalSectionHolder holder(L2BufferCS);
// ... 解码操作 ...
{
    tTJSCriticalSectionHolder holder(L2BufferRemainCS);
    L2BufferRemain++;
}
```

<details>
<summary>查看答案</summary>

两个锁有不同的职责和持有时间：

1. **L2BufferCS**：保护整个 L2 缓冲区的内容，锁持有时间长（整个解码过程）。解码线程写入数据时需要持有此锁。

2. **L2BufferRemainCS**：只保护 `L2BufferRemain` 计数器，锁持有时间很短。播放线程需要频繁读取此计数器来判断是否有数据可读。

如果只用一个锁，播放线程在检查 `L2BufferRemain` 时会阻塞解码操作，导致不必要的性能损失。使用细粒度锁可以让：
- 解码线程在解码过程中不阻塞计数器的读取
- 播放线程快速检查计数器而不等待解码完成

这是典型的**读写分离**优化策略。

</details>
