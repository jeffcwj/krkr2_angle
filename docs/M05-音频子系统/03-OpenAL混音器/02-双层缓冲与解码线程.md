## 为什么需要双层缓冲

单层缓冲的问题在于：解码和播放共用同一个缓冲区，当解码速度跟不上播放速度时会产生卡顿（buffer underrun）。

```
单层缓冲问题：
┌─────────────────────────────────────────────┐
│            SoundBuffer (L1)                 │
│  ┌───┬───┬───┬───┬───┬───┬───┬───┐         │
│  │ P │ P │ ? │ ? │ D │ D │ D │ D │         │
│  └───┴───┴───┴───┴───┴───┴───┴───┘         │
│    ↑               ↑                        │
│  播放位置        解码位置                    │
│                                             │
│  P = 已播放  ? = 即将播放  D = 已解码        │
└─────────────────────────────────────────────┘

问题：如果解码器处理复杂音频（如 Opus、Vorbis）时卡顿，
     播放位置可能追上解码位置，导致爆音或静音
```

**KrKr2 的解决方案：L1/L2 双层缓冲**

```
双层缓冲架构：
┌─────────────────────────────────────────────────────────┐
│  Level 2 Buffer (L2) - 解码缓冲区                        │
│  ┌───┬───┬───┬───┬───┬───┬───┬───┬───┬───┬───┬───┐    │
│  │ R │ R │ R │ R │ W │ W │ W │   │   │   │   │   │    │
│  └───┴───┴───┴───┴───┴───┴───┴───┴───┴───┴───┴───┘    │
│    ↑               ↑                                   │
│  读取位置        写入位置                               │
│                                                         │
│  ↓ 搬运数据                                             │
│                                                         │
│  Level 1 Buffer (L1) - 播放缓冲区                        │
│  ┌───┬───┬───┬───┐                                     │
│  │ Q │ Q │ Q │   │  → OpenAL/Oboe                      │
│  └───┴───┴───┴───┘                                     │
│    ↑           ↑                                        │
│  播放位置    填充位置                                    │
└─────────────────────────────────────────────────────────┘

R = 已读取（待搬运到 L1）
W = 已写入（解码完成）
Q = 已入队（等待 OpenAL 播放）
```

**双层缓冲的优势：**

| 特性 | 单层缓冲 | 双层缓冲 |
|------|----------|----------|
| 解码与播放耦合 | 强耦合 | 解耦 |
| 抗卡顿能力 | 弱（几十 ms 裕量） | 强（秒级裕量） |
| 内存占用 | 低 | 稍高（L2 约 1 秒数据） |
| 实现复杂度 | 简单 | 中等 |
| 适用场景 | 简单音频 | 复杂解码、网络流 |

### 主循环：动态优先级调整

```cpp
// 文件: cpp/core/sound/win32/WaveImpl.cpp 第 1828-1868 行
#define TVP_WSB_DECODE_THREAD_SLEEP_TIME 110  // 缓冲区满时休眠时间

void tTVPWaveSoundBufferDecodeThread::Execute() {
    while(!GetTerminated()) {
        DWORD st = TVPGetTickCount();
        
        while(Running) {
            bool wait;
            DWORD et;

            if(Running) {
                // 进入临界区，防止与主线程竞争
                volatile tTJSCriticalSectionHolder cs_holder(OneLoopCS);
                wait = !Owner->FillL2Buffer(false, true);  // 填充 L2 缓冲区
            }

            if(GetTerminated()) break;

            if(Running) {
                et = TVPGetTickCount();
                TVPPushEnvironNoise(&et, sizeof(et));  // 熵收集
                
                if(wait) {
                    // L2 缓冲区已满，长睡眠
                    DWORD elapsed = et - st;
                    if(elapsed < TVP_WSB_DECODE_THREAD_SLEEP_TIME) {
                        Event.WaitFor(TVP_WSB_DECODE_THREAD_SLEEP_TIME - elapsed);
                    }
                } else {
                    // L2 缓冲区未满，短睡眠并降低优先级
                    TVPRelinquishCPU();
                    if(!GetTerminated())
                        SetPriority(TVPDecodeThreadLowPriority);  // 降低优先级
                }
                st = et;
            }
        }
        
        if(GetTerminated()) break;
        Event.WaitFor(0);  // 等待 Continue() 唤醒
    }
}
```

**动态优先级机制：**

```
┌─────────────────────────────────────────────────────────────┐
│                    解码线程状态机                            │
│                                                             │
│  ┌─────────┐   Continue()   ┌─────────┐                    │
│  │ WAITING │ ──────────────→│ RUNNING │                    │
│  │(低优先级)│                │(高优先级)│                    │
│  └────┬────┘                └────┬────┘                    │
│       ↑                          │                          │
│       │      Interrupt()         ↓                          │
│       └──────────────────────────┤                          │
│                                  │                          │
│  Running 时：                    │                          │
│  - L2 未满 → 高速解码，降为低优先级                           │
│  - L2 已满 → 休眠 110ms，保持低优先级                         │
│                                                             │
│  L2 剩余 < 1秒 时：主线程会将优先级提升回 ttpNormal           │
└─────────────────────────────────────────────────────────────┘
```

```cpp
void tTVPWaveSoundBufferDecodeThread::Interrupt() {
    if(!Running) return;
    SetPriority(TVPDecodeThreadHighPriority);  // 提升优先级以快速响应
    Event.Set();
    tTJSCriticalSectionHolder cs_holder(OneLoopCS);  // 等待当前循环完成
    Running = false;
}

void tTVPWaveSoundBufferDecodeThread::Continue() {
    SetPriority(TVPDecodeThreadHighPriority);  // 重新提升优先级
    Running = true;
    Event.Set();  // 唤醒线程
}
```

## L2 缓冲区填充流程

### FillL2Buffer：核心解码方法

```cpp
// 简化版 FillL2Buffer 逻辑
bool tTJSNI_WaveSoundBuffer::FillL2Buffer(bool firstwrite, bool fromdecodethread) {
    // 检查 L2 是否已满
    {
        tTJSCriticalSectionHolder holder(L2BufferRemainCS);
        if(L2BufferRemain >= L2BufferUnits - 1) {
            return false;  // 已满，返回 false 表示需要等待
        }
    }
    
    // 获取当前写入位置
    tjs_uint8 *buffer = Level2Buffer + L2BufferWritePos * L2AccessUnitBytes;
    
    // 调用解码器渲染 PCM 数据
    tjs_uint decoded = 0;
    tTVPWaveSegmentQueue &segments = L2BufferSegmentQueues[L2BufferWritePos];
    
    if(LoopManager) {
        // 使用循环管理器解码（处理循环点、标签等）
        decoded = LoopManager->Render(buffer, AccessUnitSamples, segments);
    } else if(Decoder) {
        // 直接调用解码器
        bool cont = Decoder->Render(buffer, AccessUnitSamples, decoded);
        if(!cont) {
            L2BufferEnded = true;  // 解码结束
        }
    }
    
    // 保存解码采样数
    L2BufferDecodedSamplesInUnit[L2BufferWritePos] = decoded;
    
    // 移动写入位置（环形缓冲区）
    L2BufferWritePos++;
    if(L2BufferWritePos >= L2BufferUnits) {
        L2BufferWritePos = 0;
    }
    
    // 增加剩余数据计数
    {
        tTJSCriticalSectionHolder holder(L2BufferRemainCS);
        L2BufferRemain++;
    }
    
    return true;  // 返回 true 表示还可以继续填充
}
```

### L2 到 L1 的数据搬运

```cpp
// 文件: cpp/core/sound/win32/WaveImpl.cpp 第 2528-2576 行
void tTJSNI_WaveSoundBuffer::FillDSBuffer(tjs_int writepos,
                                          tTVPWaveSegmentQueue &segments) {
    segments.Clear();

    if(SoundBuffer->IsBufferValid()) {
        // 从 L2 读取数据
        tjs_uint decoded = ReadL2Buffer(
            UseVisBuffer ? VisBuffer + writepos : nullptr, 
            segments);
        // 数据已自动发送到 OpenAL 队列
    }
}

// ReadL2Buffer 从 L2 缓冲区读取一个单元
tjs_uint tTJSNI_WaveSoundBuffer::ReadL2Buffer(void *buffer,
                                              tTVPWaveSegmentQueue &segments) {
    // 检查 L2 是否有数据
    int bufferremain;
    {
        tTJSCriticalSectionHolder holder(L2BufferRemainCS);
        bufferremain = L2BufferRemain;
    }
    
    if(bufferremain == 0) {
        // L2 为空，填充静音
        if(buffer) {
            TVPMakeSilentWave((tjs_uint8 *)buffer, AccessUnitSamples, &InputFormat);
        }
        return 0;
    }
    
    // 读取 L2 数据
    tjs_uint decoded = L2BufferDecodedSamplesInUnit[L2BufferReadPos];
    tjs_uint8 *src = Level2Buffer + L2BufferReadPos * L2AccessUnitBytes;
    
    // 复制数据到 OpenAL 缓冲区
    if(buffer) {
        memcpy(buffer, src, decoded * InputFormat.BytesPerSample * InputFormat.Channels);
    }
    
    // 同时发送到 OpenAL
    SoundBuffer->AppendBuffer(src, decoded * InputFormat.BytesPerSample * InputFormat.Channels);
    
    // 复制片段信息（用于标签事件）
    segments = L2BufferSegmentQueues[L2BufferReadPos];
    
    // 移动读取位置
    L2BufferReadPos++;
    if(L2BufferReadPos >= L2BufferUnits) {
        L2BufferReadPos = 0;
    }
    
    // 减少剩余数据计数
    {
        tTJSCriticalSectionHolder holder(L2BufferRemainCS);
        L2BufferRemain--;
    }
    
    return decoded;
}
```

## 主线程与解码线程协作

### 启动播放流程

```cpp
// 文件: cpp/core/sound/win32/WaveImpl.cpp 第 2892-2940 行
void tTJSNI_WaveSoundBuffer::StartPlay() {
    if(!Decoder) return;

    TVPEnsurePrimaryBufferPlay();       // 确保主缓冲区运行
    TVPEnsureWaveSoundBufferWorking();  // 确保工作线程运行

    { // 线程保护块
        if(Thread->GetRunning()) {
            Thread->SetPriority(TVPDecodeThreadHighPriority);
        }
        tTJSCriticalSectionHolder holder(BufferCS);
        tTJSCriticalSectionHolder l2holder(L2BufferCS);

        CreateSoundBuffer();   // 创建 OpenAL 缓冲区
        ResetFilterChain();    // 重置 DSP 滤波器

        // 预填充：先填 L2，再填 L1
        BufferPlaying = true;
        FillL2Buffer(true, false);      // 填充整个 L2（阻塞）
        FillBuffer(true, false);        // 填充 L1 第一个单元
        FillBuffer(false, false);       // 填充 L1 第二个单元
        FillBuffer(false, false);       // 填充 L1 第三个单元
        FillBuffer(false, false);       // 填充 L1 第四个单元

        // 开始播放
        if(!Paused) {
            SoundBuffer->Play();
            DSBufferPlaying = true;
        }

        ResetLastCheckedDecodePos();
        TVPReschedulePendingLabelEvent(GetNearestEventStep());
    } // 结束线程保护块

    TVPEnsureWaveSoundBufferWorking();  // 再次唤醒工作线程
    ThreadCallbackEnabled = true;
    Thread->Continue();                 // 启动解码线程持续填充
}
```

## 动手实践：简化版双层缓冲播放器

```cpp
// dual_buffer_player.cpp
#include <AL/al.h>
#include <AL/alc.h>
#include <atomic>
#include <chrono>
#include <cmath>
#include <condition_variable>
#include <cstring>
#include <iostream>
#include <mutex>
#include <queue>
#include <thread>
#include <vector>

// 音频格式参数
constexpr int SAMPLE_RATE = 48000;
constexpr int CHANNELS = 2;
constexpr int BITS_PER_SAMPLE = 16;
constexpr int FRAME_SIZE = CHANNELS * (BITS_PER_SAMPLE / 8);

// 缓冲区参数
constexpr int L1_BUFFER_COUNT = 4;          // L1 缓冲区数量
constexpr int L2_UNIT_SAMPLES = 480;        // 每单元采样数（10ms）
constexpr int L2_UNIT_COUNT = 100;          // L2 单元数量（1秒）
constexpr int L2_UNIT_BYTES = L2_UNIT_SAMPLES * FRAME_SIZE;

// L2 缓冲区单元
struct L2Unit {
    std::vector<int16_t> data;
    int sampleCount = 0;
    bool valid = false;
    
    L2Unit() : data(L2_UNIT_SAMPLES * CHANNELS) {}
};

// 双层缓冲音频播放器
class DualBufferPlayer {
    // OpenAL 资源
    ALCdevice* device_ = nullptr;
    ALCcontext* context_ = nullptr;
    ALuint source_;
    ALuint buffers_[L1_BUFFER_COUNT];
    
    // L2 缓冲区（环形）
    std::vector<L2Unit> l2Buffer_;
    std::atomic<int> l2ReadPos_{0};
    std::atomic<int> l2WritePos_{0};
    std::atomic<int> l2Remain_{0};
    
    // 解码线程
    std::thread decodeThread_;
    std::atomic<bool> running_{false};
    std::atomic<bool> terminated_{false};
    std::mutex decodeMutex_;
    std::condition_variable decodeCV_;
    
    // 模拟解码器（生成正弦波）
    float phase_ = 0.0f;
    float frequency_ = 440.0f;
    
public:
    DualBufferPlayer() : l2Buffer_(L2_UNIT_COUNT) {
        // 初始化 OpenAL
        device_ = alcOpenDevice(nullptr);
        context_ = alcCreateContext(device_, nullptr);
        alcMakeContextCurrent(context_);
        
        alGenSources(1, &source_);
        alGenBuffers(L1_BUFFER_COUNT, buffers_);
        
        std::cout << "[DualBufferPlayer] 初始化完成\n";
        std::cout << "  L1: " << L1_BUFFER_COUNT << " 缓冲区\n";
        std::cout << "  L2: " << L2_UNIT_COUNT << " 单元 × " 
                  << L2_UNIT_SAMPLES << " 采样 = " 
                  << (L2_UNIT_COUNT * L2_UNIT_SAMPLES / SAMPLE_RATE * 1000) << "ms\n";
    }
    
    ~DualBufferPlayer() {
        stop();
        alDeleteBuffers(L1_BUFFER_COUNT, buffers_);
        alDeleteSources(1, &source_);
        alcMakeContextCurrent(nullptr);
        alcDestroyContext(context_);
        alcCloseDevice(device_);
    }
    
    // 启动解码线程
    void startDecodeThread() {
        terminated_ = false;
        decodeThread_ = std::thread([this]() {
            std::cout << "[解码线程] 启动\n";
            
            while (!terminated_) {
                std::unique_lock<std::mutex> lock(decodeMutex_);
                
                if (!running_) {
                    // 等待 Continue() 信号
                    decodeCV_.wait(lock, [this]() { 
                        return running_ || terminated_; 
                    });
                    continue;
                }
                
                lock.unlock();
                
                // 填充 L2 缓冲区
                while (running_ && !terminated_) {
                    bool filled = fillL2Buffer();
                    if (!filled) {
                        // L2 已满，休眠
                        std::this_thread::sleep_for(std::chrono::milliseconds(50));
                    }
                }
            }
            
            std::cout << "[解码线程] 退出\n";
        });
    }
    
    // 填充 L2 缓冲区
    bool fillL2Buffer() {
        if (l2Remain_ >= L2_UNIT_COUNT - 1) {
            return false;  // L2 已满
        }
        
        int writePos = l2WritePos_;
        L2Unit& unit = l2Buffer_[writePos];
        
        // 模拟解码：生成正弦波
        for (int i = 0; i < L2_UNIT_SAMPLES; ++i) {
            float sample = std::sin(2.0f * 3.14159f * phase_);
            int16_t pcm = static_cast<int16_t>(sample * 16000);
            unit.data[i * 2] = pcm;      // 左声道
            unit.data[i * 2 + 1] = pcm;  // 右声道
            phase_ += frequency_ / SAMPLE_RATE;
            if (phase_ >= 1.0f) phase_ -= 1.0f;
        }
        
        unit.sampleCount = L2_UNIT_SAMPLES;
        unit.valid = true;
        
        // 移动写入位置
        l2WritePos_ = (writePos + 1) % L2_UNIT_COUNT;
        l2Remain_++;
        
        return true;
    }
    
    // 从 L2 读取数据到 L1
    int readL2Buffer(int16_t* dest) {
        if (l2Remain_ <= 0) {
            // L2 为空，填充静音
            std::memset(dest, 0, L2_UNIT_BYTES);
            return 0;
        }
        
        int readPos = l2ReadPos_;
        L2Unit& unit = l2Buffer_[readPos];
        
        std::memcpy(dest, unit.data.data(), unit.sampleCount * FRAME_SIZE);
        int samples = unit.sampleCount;
        
        unit.valid = false;
        l2ReadPos_ = (readPos + 1) % L2_UNIT_COUNT;
        l2Remain_--;
        
        return samples;
    }
    
    // 填充 L1 缓冲区
    void fillL1Buffer(ALuint buffer) {
        std::vector<int16_t> pcmData(L2_UNIT_SAMPLES * CHANNELS);
        int samples = readL2Buffer(pcmData.data());
        
        alBufferData(buffer, AL_FORMAT_STEREO16, pcmData.data(),
                     samples * FRAME_SIZE, SAMPLE_RATE);
        alSourceQueueBuffers(source_, 1, &buffer);
    }
    
    // 开始播放
    void play() {
        // 重置状态
        l2ReadPos_ = 0;
        l2WritePos_ = 0;
        l2Remain_ = 0;
        phase_ = 0.0f;
        
        // 启动解码线程
        startDecodeThread();
        
        // 预填充 L2
        running_ = true;
        decodeCV_.notify_one();
        
        std::cout << "[主线程] 预填充 L2...\n";
        while (l2Remain_ < L2_UNIT_COUNT / 2) {
            std::this_thread::sleep_for(std::chrono::milliseconds(10));
        }
        std::cout << "[主线程] L2 已填充 " << l2Remain_.load() << " 单元\n";
        
        // 预填充 L1
        for (int i = 0; i < L1_BUFFER_COUNT; ++i) {
            fillL1Buffer(buffers_[i]);
        }
        std::cout << "[主线程] L1 已填充 " << L1_BUFFER_COUNT << " 缓冲区\n";
        
        // 开始播放
        alSourcePlay(source_);
        std::cout << "[主线程] 播放开始\n";
        
        // 持续填充 L1
        while (running_) {
            ALint processed = 0;
            alGetSourcei(source_, AL_BUFFERS_PROCESSED, &processed);
            
            while (processed > 0) {
                ALuint buffer;
                alSourceUnqueueBuffers(source_, 1, &buffer);
                fillL1Buffer(buffer);
                processed--;
            }
            
            // 检查 L2 剩余量
            if (l2Remain_ < 20) {
                std::cout << "[警告] L2 剩余不足: " << l2Remain_.load() << "\n";
            }
            
            std::this_thread::sleep_for(std::chrono::milliseconds(10));
        }
    }
    
    // 停止播放
    void stop() {
        running_ = false;
        terminated_ = true;
        decodeCV_.notify_one();
        
        if (decodeThread_.joinable()) {
            decodeThread_.join();
        }
        
        alSourceStop(source_);
        
        // 清空队列
        ALint queued;
        alGetSourcei(source_, AL_BUFFERS_QUEUED, &queued);
        while (queued-- > 0) {
            ALuint buffer;
            alSourceUnqueueBuffers(source_, 1, &buffer);
        }
    }
    
    // 设置频率（模拟解码器参数变化）
    void setFrequency(float freq) {
        frequency_ = freq;
    }
};

int main() {
    DualBufferPlayer player;
    
    std::cout << "\n按 Enter 开始播放，再按 Enter 停止...\n";
    std::cin.get();
    
    // 在后台线程播放
    std::thread playThread([&player]() {
        player.play();
    });
    
    // 动态改变频率
    for (int i = 0; i < 10; ++i) {
        std::this_thread::sleep_for(std::chrono::milliseconds(500));
        float freq = 440.0f + i * 50.0f;
        player.setFrequency(freq);
        std::cout << "频率: " << freq << " Hz\n";
    }
    
    std::cin.get();
    player.stop();
    
    if (playThread.joinable()) {
        playThread.join();
    }
    
    std::cout << "播放结束\n";
    return 0;
}
```

## 本节小结

- KrKr2 采用 L1/L2 双层缓冲架构，将解码与播放解耦
- L1（约 40ms）是 OpenAL 队列缓冲区，L2（约 1 秒）是解码缓冲区
- 解码线程持续填充 L2，主/工作线程从 L2 搬运到 L1
- 动态优先级机制：L2 满时降低优先级，L2 不足时提升优先级
- 双层缓冲可有效避免因解码卡顿导致的音频爆音

## 练习题与答案

### 题目 1：计算缓冲延迟

假设采样率 48000Hz，L1 为 4 个 480 采样的缓冲区，L2 为 100 个单元。计算最大端到端延迟。

<details>
<summary>查看答案</summary>

**计算过程：**

1. **L1 延迟（最坏情况：所有缓冲区都已入队等待播放）**
   - L1 总采样数 = 4 × 480 = 1920 samples
   - L1 延迟 = 1920 / 48000 = 0.04 秒 = **40ms**

2. **L2 延迟（最坏情况：L2 已满）**
   - L2 总采样数 = 100 × 480 = 48000 samples
   - L2 延迟 = 48000 / 48000 = 1.0 秒 = **1000ms**

3. **OpenAL 硬件延迟（典型值）**
   - 约 **10-50ms**（取决于音频驱动和缓冲配置）

**最大端到端延迟 = L1 + L2 + 硬件 ≈ 40 + 1000 + 30 = 1070ms ≈ 1.1 秒**

**实际延迟（典型情况）：**
- L2 通常不会完全填满，维持在 50% 左右
- 实际延迟约 40 + 500 + 30 = **570ms**

对于视觉小说游戏，这个延迟完全可以接受。对于需要低延迟的场景（如节奏游戏），应减少 L2 长度。

</details>

### 题目 2：解释为什么使用两个临界区

代码中使用了 `BufferCS` 和 `L2BufferCS` 两个临界区，为什么不合并为一个？

<details>
<summary>查看答案</summary>

**分离临界区的原因：**

1. **减少锁竞争**
   - `L2BufferCS` 保护 L2 缓冲区操作
   - `BufferCS` 保护 L1 和整体状态
   - 解码线程主要访问 L2，主线程主要访问 L1
   - 分离后两者可以并行工作

2. **避免死锁**
   - 如果只用一个锁，解码线程填充 L2 时会阻塞主线程从 L2 读取
   - 分离后，只有在 L2 数据不足时才需要同时持有两个锁

3. **锁的粒度原则**
   - 细粒度锁提高并发性
   - 粗粒度锁简化代码但降低性能

**锁的获取顺序（防止死锁）：**
```cpp
// 正确顺序：先 BufferCS，后 L2BufferCS
tTJSCriticalSectionHolder holder(BufferCS);
tTJSCriticalSectionHolder l2holder(L2BufferCS);
```

始终按相同顺序获取锁可以防止死锁。

</details>

### 题目 3：实现 L2 缓冲区监控

编写一个函数，返回 L2 缓冲区的使用率百分比和预估剩余播放时间。

<details>
<summary>查看答案</summary>

```cpp
struct L2BufferStatus {
    float usagePercent;        // 使用率 (0-100%)
    float remainingSeconds;    // 剩余播放时间（秒）
    int remainingUnits;        // 剩余单元数
    int totalUnits;            // 总单元数
};

L2BufferStatus getL2BufferStatus() {
    L2BufferStatus status;
    
    // 获取当前状态（需要加锁）
    int remain;
    {
        tTJSCriticalSectionHolder holder(L2BufferRemainCS);
        remain = L2BufferRemain;
    }
    
    status.remainingUnits = remain;
    status.totalUnits = L2BufferUnits;
    
    // 计算使用率
    status.usagePercent = (float)remain / L2BufferUnits * 100.0f;
    
    // 计算剩余播放时间
    // 每个单元 = AccessUnitSamples 个采样
    // 总剩余采样 = remain * AccessUnitSamples
    // 剩余时间 = 总剩余采样 / 采样率
    int remainingSamples = remain * AccessUnitSamples;
    status.remainingSeconds = (float)remainingSamples / InputFormat.SamplesPerSec;
    
    return status;
}

// 使用示例
void monitorL2Buffer() {
    while (playing) {
        L2BufferStatus status = getL2BufferStatus();
        
        std::cout << "L2 状态: " 
                  << status.remainingUnits << "/" << status.totalUnits
                  << " (" << status.usagePercent << "%)"
                  << ", 剩余 " << status.remainingSeconds << " 秒\n";
        
        // 低水位警告
        if (status.usagePercent < 20.0f) {
            std::cout << "[警告] L2 缓冲区即将耗尽！\n";
            // 可以提升解码线程优先级
        }
        
        std::this_thread::sleep_for(std::chrono::milliseconds(500));
    }
}
```
