## 5.6 Creator 注册机制回顾

### 5.6.1 注册链架构

KrKr2 使用 **Creator 链** 管理所有音频解码器：

```
TVPCreateWaveDecoder(storagename)
              │
              ▼
   ┌────────────────────────────────────────┐
   │  tTVPWaveDecoderManager::Creators      │
   │  (std::vector<tTVPWaveDecoderCreator*>)│
   ├────────────────────────────────────────┤
   │  [0] FFWaveDecoderCreator              │◄─ 最后注册，最先尝试
   │  [1] OpusWaveDecoderCreator            │
   │  [2] RIFFWaveDecoderCreator (.wav)     │
   │  [3] VorbisWaveDecoderCreator (.ogg)   │◄─ 最先注册，最后尝试
   │  [4] FlacWaveDecoderCreator (.flac) ←──│─── 我们要添加的
   └────────────────────────────────────────┘
              │
              ▼ (逆序遍历)
   for(i = size-1; i >= 0; i--)
       decoder = Creators[i]->Create(name, ext)
       if(decoder) return decoder;
```

**关键点：**
- 注册顺序决定尝试顺序（后注册先尝试）
- 每个 Creator 检查扩展名，不匹配返回 `nullptr`
- 第一个成功创建的解码器被使用

### 5.6.2 注册时机

解码器在 `tTVPWaveDecoderManager` 构造函数中注册：

```cpp
// cpp/core/sound/WaveIntf.cpp 第 712-729 行
struct tTVPWaveDecoderManager {
    std::vector<tTVPWaveDecoderCreator *> Creators;
    tTVPWDC_RIFFWave RIFFWaveDecoderCreator;
    VorbisWaveDecoderCreator vorbisWaveDecoderCreator;
    FFWaveDecoderCreator ffWaveDecoderCreator;
    OpusWaveDecoderCreator opusWaveDecoderCreator;
    // 添加 FLAC Creator
    FlacWaveDecoderCreator flacWaveDecoderCreator;  // ← 新增

    tTVPWaveDecoderManager() {
        TVPWaveDecoderManagerAvail = true;
        TVPRegisterWaveDecoderCreator(&ffWaveDecoderCreator);
        TVPRegisterWaveDecoderCreator(&opusWaveDecoderCreator);
        TVPRegisterWaveDecoderCreator(&RIFFWaveDecoderCreator);
        TVPRegisterWaveDecoderCreator(&vorbisWaveDecoderCreator);
        TVPRegisterWaveDecoderCreator(&flacWaveDecoderCreator);  // ← 新增
    }
};
```

### 5.7.2 Creator 类实现

确保 `FlacWaveDecoderCreator::Create` 正确实现：

```cpp
// cpp/core/sound/FlacWaveDecoder.cpp

tTVPWaveDecoder *FlacWaveDecoderCreator::Create(
    const ttstr &storagename, const ttstr &extension) {
    
    // 支持 .flac 和 .fla 两种扩展名
    static const ttstr flac_ext = TJS_W(".flac");
    static const ttstr fla_ext = TJS_W(".fla");
    
    // 快速扩展名检查（大小写不敏感已由调用方处理）
    if(extension != flac_ext && extension != fla_ext) {
        return nullptr;  // 不是 FLAC 文件，让下一个 Creator 尝试
    }
    
    // 尝试创建解码器
    FlacWaveDecoder *decoder = new FlacWaveDecoder();
    
    if(!decoder->SetStream(storagename)) {
        // 打开或初始化失败
        delete decoder;
        return nullptr;
    }
    
    return decoder;
}
```

**扩展名处理说明：**

```cpp
// TVPCreateWaveDecoder 中已将扩展名转为小写
// cpp/core/sound/WaveIntf.cpp 第 756-758 行
ttstr ext(TVPExtractStorageExt(storagename));
ext.ToLowerCase();  // ← 转小写

// 因此 Creator 只需比较小写扩展名
```

### 5.8.3 多平台编译注意事项

| 平台 | libFLAC 包名 | 链接目标 | 注意事项 |
|------|-------------|---------|---------|
| Windows | `flac` | `FLAC::FLAC` | 自动处理静态/动态链接 |
| Linux | `flac` | `FLAC::FLAC` | 需要 `libogg` 依赖 |
| macOS | `flac` | `FLAC::FLAC` | 同 Linux |
| Android | `flac` | `FLAC::FLAC` | vcpkg_android.cmake 自动配置 |

**Android 特殊处理：**

```cmake
# cpp/core/sound/CMakeLists.txt Android 部分
if(ANDROID)
    # Android 上需要显式链接 ogg（FLAC 依赖）
    find_package(Ogg CONFIG REQUIRED)
    target_link_libraries(core_sound_module
        PRIVATE
            Ogg::ogg
    )
```

### 5.9.2 完整测试代码

```cpp
// tests/unit-tests/core/sound/test_flac_decoder.cpp

#include <catch2/catch_all.hpp>
#include "FlacWaveDecoder.h"
#include "StorageIntf.h"
#include <cmath>
#include <vector>

// 测试文件路径（由 CMake configure_file 生成）
static const char *TEST_FILES_PATH = TEST_FILES_PATH_STR;

// 辅助函数：构建完整测试文件路径
static ttstr GetTestFilePath(const char *filename) {
    return ttstr(TEST_FILES_PATH) + TJS_W("/audio/") + ttstr(filename);
}

TEST_CASE("FlacWaveDecoderCreator extension check", "[sound][flac]") {
    FlacWaveDecoderCreator creator;
    
    SECTION("accepts .flac extension") {
        ttstr ext = TJS_W(".flac");
        // Create 会因为文件不存在而返回 nullptr，但不会崩溃
        // 这里我们只测试扩展名逻辑
        REQUIRE_NOTHROW(creator.Create(TJS_W("dummy.flac"), ext));
    }
    
    SECTION("accepts .fla extension") {
        ttstr ext = TJS_W(".fla");
        REQUIRE_NOTHROW(creator.Create(TJS_W("dummy.fla"), ext));
    }
    
    SECTION("rejects .mp3 extension") {
        ttstr ext = TJS_W(".mp3");
        tTVPWaveDecoder *decoder = creator.Create(TJS_W("dummy.mp3"), ext);
        REQUIRE(decoder == nullptr);
    }
    
    SECTION("rejects .ogg extension") {
        ttstr ext = TJS_W(".ogg");
        tTVPWaveDecoder *decoder = creator.Create(TJS_W("dummy.ogg"), ext);
        REQUIRE(decoder == nullptr);
    }
}

TEST_CASE("FlacWaveDecoder format parsing", "[sound][flac]") {
    FlacWaveDecoderCreator creator;
    
    SECTION("44.1kHz stereo 16-bit") {
        ttstr path = GetTestFilePath("test_44100_stereo.flac");
        tTVPWaveDecoder *decoder = creator.Create(path, TJS_W(".flac"));
        
        if(decoder) {  // 文件可能不存在
            tTVPWaveFormat format;
            decoder->GetFormat(format);
            
            CHECK(format.SamplesPerSec == 44100);
            CHECK(format.Channels == 2);
            CHECK(format.BitsPerSample == 16);
            CHECK(format.IsFloat == false);
            CHECK(format.TotalSamples > 0);
            
            delete decoder;
        } else {
            WARN("Test file not found: test_44100_stereo.flac");
        }
    }
    
    SECTION("48kHz mono") {
        ttstr path = GetTestFilePath("test_48000_mono.flac");
        tTVPWaveDecoder *decoder = creator.Create(path, TJS_W(".flac"));
        
        if(decoder) {
            tTVPWaveFormat format;
            decoder->GetFormat(format);
            
            CHECK(format.SamplesPerSec == 48000);
            CHECK(format.Channels == 1);
            
            delete decoder;
        }
    }
    
    SECTION("96kHz 24-bit") {
        ttstr path = GetTestFilePath("test_96000_24bit.flac");
        tTVPWaveDecoder *decoder = creator.Create(path, TJS_W(".flac"));
        
        if(decoder) {
            tTVPWaveFormat format;
            decoder->GetFormat(format);
            
            CHECK(format.SamplesPerSec == 96000);
            // 注意：解码器输出 16 位，但原始位深度信息保留
            CHECK(format.BitsPerSample == 24);
            
            delete decoder;
        }
    }
}

TEST_CASE("FlacWaveDecoder rendering", "[sound][flac]") {
    FlacWaveDecoderCreator creator;
    ttstr path = GetTestFilePath("test_44100_stereo.flac");
    tTVPWaveDecoder *decoder = creator.Create(path, TJS_W(".flac"));
    
    if(!decoder) {
        WARN("Test file not found, skipping render tests");
        return;
    }
    
    tTVPWaveFormat format;
    decoder->GetFormat(format);
    
    SECTION("render first frame") {
        // 分配缓冲区（1024 样本 × 通道数 × 2 字节）
        std::vector<tjs_int16> buffer(1024 * format.Channels);
        tjs_uint rendered = 0;
        
        bool ok = decoder->Render(buffer.data(), 1024, rendered);
        
        // 应该成功渲染一些样本
        REQUIRE(rendered > 0);
        REQUIRE(rendered <= 1024);
        
        // 检查数据不全为零（除非是静音测试文件）
        bool hasNonZero = false;
        for(size_t i = 0; i < rendered * format.Channels; i++) {
            if(buffer[i] != 0) {
                hasNonZero = true;
                break;
            }
        }
        // 如果是正常音频文件，应该有非零样本
        // （静音文件可能全零，此检查仅供参考）
        INFO("Rendered " << rendered << " samples");
    }
    
    SECTION("render entire file") {
        std::vector<tjs_int16> buffer(4096 * format.Channels);
        tjs_uint64 totalRendered = 0;
        tjs_uint rendered;
        
        while(decoder->Render(buffer.data(), 4096, rendered)) {
            totalRendered += rendered;
            if(rendered < 4096) break;  // 文件结束
        }
        totalRendered += rendered;  // 加上最后一次
        
        // 总渲染样本数应该等于或接近文件总样本数
        CHECK(totalRendered > 0);
        if(format.TotalSamples > 0) {
            CHECK(totalRendered == format.TotalSamples);
        }
    }
    
    delete decoder;
}

TEST_CASE("FlacWaveDecoder seeking", "[sound][flac]") {
    FlacWaveDecoderCreator creator;
    ttstr path = GetTestFilePath("test_44100_stereo.flac");
    tTVPWaveDecoder *decoder = creator.Create(path, TJS_W(".flac"));
    
    if(!decoder) {
        WARN("Test file not found, skipping seek tests");
        return;
    }
    
    tTVPWaveFormat format;
    decoder->GetFormat(format);
    
    SECTION("seek to beginning") {
        // 先读一些数据
        std::vector<tjs_int16> buffer(1024 * format.Channels);
        tjs_uint rendered;
        decoder->Render(buffer.data(), 1024, rendered);
        
        // Seek 回开头
        bool seekOk = decoder->SetPosition(0);
        REQUIRE(seekOk);
        
        // 再次读取，应该得到相同数据
        std::vector<tjs_int16> buffer2(1024 * format.Channels);
        tjs_uint rendered2;
        decoder->Render(buffer2.data(), 1024, rendered2);
        
        REQUIRE(rendered == rendered2);
        // 比较前 100 个样本
        for(size_t i = 0; i < std::min(100u, rendered * format.Channels); i++) {
            CHECK(buffer[i] == buffer2[i]);
        }
    }
    
    SECTION("seek to middle") {
        if(format.TotalSamples > 10000) {
            tjs_uint64 midPoint = format.TotalSamples / 2;
            bool seekOk = decoder->SetPosition(midPoint);
            REQUIRE(seekOk);
            
            // 应该能继续读取
            std::vector<tjs_int16> buffer(1024 * format.Channels);
            tjs_uint rendered;
            decoder->Render(buffer.data(), 1024, rendered);
            REQUIRE(rendered > 0);
        }
    }
    
    SECTION("seek past end fails") {
        if(format.TotalSamples > 0) {
            bool seekOk = decoder->SetPosition(format.TotalSamples + 1000);
            CHECK_FALSE(seekOk);
        }
    }
    
    delete decoder;
}

TEST_CASE("FlacWaveDecoder stress test", "[sound][flac][stress]") {
    FlacWaveDecoderCreator creator;
    ttstr path = GetTestFilePath("test_44100_stereo.flac");
    tTVPWaveDecoder *decoder = creator.Create(path, TJS_W(".flac"));
    
    if(!decoder) {
        WARN("Test file not found, skipping stress tests");
        return;
    }
    
    tTVPWaveFormat format;
    decoder->GetFormat(format);
    
    SECTION("random seeks") {
        std::vector<tjs_int16> buffer(256 * format.Channels);
        tjs_uint rendered;
        
        // 随机 Seek 100 次
        for(int i = 0; i < 100; i++) {
            tjs_uint64 pos = (rand() % format.TotalSamples);
            decoder->SetPosition(pos);
            decoder->Render(buffer.data(), 256, rendered);
        }
        
        // 能到这里说明没有崩溃
        SUCCEED();
    }
    
    SECTION("small buffer reads") {
        // 用非常小的缓冲区读取
        std::vector<tjs_int16> buffer(16 * format.Channels);
        tjs_uint rendered;
        tjs_uint64 totalRendered = 0;
        
        while(totalRendered < 10000) {
            if(!decoder->Render(buffer.data(), 16, rendered)) {
                break;
            }
            totalRendered += rendered;
        }
        
        REQUIRE(totalRendered >= 10000);
    }
    
    delete decoder;
}
```

## 5.10 调试与常见问题

### 5.10.1 常见错误排查

| 症状 | 可能原因 | 解决方案 |
|------|---------|---------|
| `undefined reference to FLAC__stream_decoder_new` | 未链接 libFLAC | 检查 CMake `target_link_libraries` |
| `Cannot open file` | 路径格式错误 | 使用 `TVPNormalizeStorageName` 规范化路径 |
| `No STREAMINFO found` | FLAC 文件损坏或不是 FLAC | 用 `flac -t` 验证文件完整性 |
| `Seek failed` | 流不支持 Seek 或位置超范围 | 检查 `Format.Seekable` 和位置范围 |
| 解码数据全零 | 位深度转换错误 | 验证 WriteCallback 中的移位逻辑 |

## 练习题与答案

### 题目 1：为什么 Creator 注册顺序是"后注册先尝试"？这对添加新格式有什么影响？

<details>
<summary>查看答案</summary>

`TVPCreateWaveDecoder` 使用逆序遍历：

```cpp
// WaveIntf.cpp 第 759-766 行
tjs_int i = (tjs_int)(TVPWaveDecoderManager.Creators.size() - 1);
for(; i >= 0; i--) {
    tTVPWaveDecoder *decoder;
    decoder = TVPWaveDecoderManager.Creators[i]->Create(storagename, ext);
    if(decoder) return decoder;
}
```

**设计意图：**
- 后注册的解码器（通常是更专业的）优先尝试
- 通用解码器（如 FFmpeg）可以放在前面作为兜底

**对添加新格式的影响：**
- 专业解码器（如 FLAC）应该**后**注册，让它优先于 FFmpeg
- 如果 FFmpeg 先匹配了 `.flac`，专业解码器就不会被使用
- 实际代码中 FFmpeg 最先注册，所以是最后尝试的兜底选项

</details>

### 题目 2：如何让 FLAC 解码器同时支持 `.flac` 和 `.FLAC`（大写）扩展名？

<details>
<summary>查看答案</summary>

不需要特殊处理。`TVPCreateWaveDecoder` 在调用 Creator 前已将扩展名转为小写：

```cpp
// WaveIntf.cpp 第 756-758 行
ttstr ext(TVPExtractStorageExt(storagename));
ext.ToLowerCase();  // ← 转小写
```

因此 Creator 只需比较小写扩展名：

```cpp
static const ttstr flac_ext = TJS_W(".flac");
if(extension != flac_ext) {  // extension 保证是小写
    return nullptr;
}
```

所有大小写变体（`.FLAC`、`.Flac`、`.fLaC`）都会被归一化为 `.flac`。

</details>

### 题目 3：编写一个测试用例，验证 FLAC 解码器在文件不存在时不会崩溃。

<details>
<summary>查看答案</summary>

```cpp
TEST_CASE("FlacWaveDecoder handles missing file gracefully", "[sound][flac]") {
    FlacWaveDecoderCreator creator;
    
    // 尝试打开不存在的文件
    ttstr nonExistent = TJS_W("this_file_does_not_exist.flac");
    ttstr ext = TJS_W(".flac");
    
    // 不应该崩溃，应该返回 nullptr
    tTVPWaveDecoder *decoder = nullptr;
    REQUIRE_NOTHROW(decoder = creator.Create(nonExistent, ext));
    
    // 应该返回 nullptr
    REQUIRE(decoder == nullptr);
    
    // 确保没有资源泄漏（decoder 已经是 nullptr，无需 delete）
}

TEST_CASE("FlacWaveDecoder handles invalid FLAC data", "[sound][flac]") {
    FlacWaveDecoderCreator creator;
    
    // 创建一个包含无效数据的临时文件
    // （实际测试中可以准备一个损坏的 FLAC 文件）
    ttstr invalidFile = TJS_W("tests/test_files/invalid.flac");
    ttstr ext = TJS_W(".flac");
    
    tTVPWaveDecoder *decoder = nullptr;
    REQUIRE_NOTHROW(decoder = creator.Create(invalidFile, ext));
    
    // 应该返回 nullptr（无法解析元数据）
    // 或者返回有效解码器但后续 Render 失败
    if(decoder) {
        std::vector<tjs_int16> buffer(1024);
        tjs_uint rendered;
        // 不应该崩溃
        REQUIRE_NOTHROW(decoder->Render(buffer.data(), 1024, rendered));
        delete decoder;
    }
}
```

关键点：
- 使用 `REQUIRE_NOTHROW` 确保不抛出异常
- 检查返回值为 `nullptr`
- 即使返回了解码器，后续操作也不应崩溃

</details>
