### 关键设计决策

| 决策项 | 选择 | 原因 |
|--------|------|------|
| 坐标系 | 屏幕坐标（左上角原点，Y 轴向下） | 2D 游戏/UI 的惯例 |
| 投影 | 正交投影 | 2D 无透视效果 |
| 顶点布局 | 交错式（位置+UV+颜色） | 缓存友好 |
| 批处理 | 相同纹理的精灵合并为一个 Draw Call | 减少状态切换 |
| 混合模式 | `GL_SRC_ALPHA, GL_ONE_MINUS_SRC_ALPHA` | 标准 Alpha 混合 |

## 第一步：着色器

### 顶点着色器

2D 渲染器的顶点着色器接收屏幕坐标，通过正交投影矩阵转换为 NDC：

```cpp
// 着色器源码（C++ 字符串）
const char* vertexShaderSrc = R"(
#version 330 core
layout(location = 0) in vec2 aPos;       // 屏幕坐标（像素）
layout(location = 1) in vec2 aTexCoord;  // 纹理坐标 [0,1]
layout(location = 2) in vec4 aColor;     // 顶点颜色（含 alpha）

uniform mat4 uProjection;  // 正交投影矩阵

out vec2 vTexCoord;
out vec4 vColor;

void main() {
    gl_Position = uProjection * vec4(aPos, 0.0, 1.0);
    vTexCoord = aTexCoord;
    vColor = aColor;
}
)";
```

### 片段着色器

```cpp
const char* fragmentShaderSrc = R"(
#version 330 core
in vec2 vTexCoord;
in vec4 vColor;

uniform sampler2D uTexture;

out vec4 FragColor;

void main() {
    // 纹理颜色 × 顶点颜色（支持颜色调制和透明度控制）
    FragColor = texture(uTexture, vTexCoord) * vColor;
}
)";
```

**为什么要乘顶点颜色？** 这样可以在不换着色器的情况下实现：
- **透明度控制**：设 `vColor.a = 0.5` 即为半透明
- **颜色调制**：设 `vColor.rgb = vec3(1,0,0)` 可将精灵染红（用于受击闪烁等效果）
- **纯白色**：`vColor = vec4(1,1,1,1)` 时不影响纹理颜色

### OpenGL ES 2.0 兼容版本

```cpp
// ES 2.0 顶点着色器
const char* vertexShaderSrc_ES2 = R"(
#version 100
attribute vec2 aPos;
attribute vec2 aTexCoord;
attribute vec4 aColor;

uniform mat4 uProjection;

varying vec2 vTexCoord;
varying vec4 vColor;

void main() {
    gl_Position = uProjection * vec4(aPos, 0.0, 1.0);
    vTexCoord = aTexCoord;
    vColor = aColor;
}
)";

// ES 2.0 片段着色器
const char* fragmentShaderSrc_ES2 = R"(
#version 100
precision mediump float;
varying vec2 vTexCoord;
varying vec4 vColor;

uniform sampler2D uTexture;

void main() {
    gl_FragColor = texture2D(uTexture, vTexCoord) * vColor;
}
)";
```

## 第二步：正交投影矩阵

2D 渲染使用正交投影，将屏幕像素坐标映射到 NDC（`[-1,1]`）：

```
屏幕坐标系:                    NDC:
(0,0)────────(W,0)           (-1,+1)────(+1,+1)
  │                │    投影     │              │
  │                │   ──→      │              │
(0,H)────────(W,H)           (-1,-1)────(+1,-1)
```

正交投影矩阵公式（左上角原点，Y 轴向下）：

```
       ┌  2/(R-L)     0         0        -(R+L)/(R-L)  ┐
 P  =  │    0      2/(T-B)      0        -(T+B)/(T-B)  │
       │    0         0       -2/(F-N)   -(F+N)/(F-N)   │
       └    0         0         0             1          ┘

其中: L=0, R=screenW, T=0, B=screenH, N=-1, F=1
```

C++ 实现：

```cpp
// 生成正交投影矩阵（列主序，OpenGL 惯例）
// left=0, right=screenW, top=0, bottom=screenH
// 这样 (0,0) 在左上角，Y 向下增长 —— 符合 2D 游戏惯例
void createOrthoProjection(float* mat, float width, float height) {
    // 列主序 4×4 矩阵（OpenGL 要求）
    //     col0        col1        col2        col3
    mat[0]  = 2.0f / width;
    mat[1]  = 0.0f;
    mat[2]  = 0.0f;
    mat[3]  = 0.0f;

    mat[4]  = 0.0f;
    mat[5]  = -2.0f / height;  // 负号：翻转 Y 轴（屏幕坐标 Y 向下）
    mat[6]  = 0.0f;
    mat[7]  = 0.0f;

    mat[8]  = 0.0f;
    mat[9]  = 0.0f;
    mat[10] = -1.0f;           // Z: near=-1, far=1
    mat[11] = 0.0f;

    mat[12] = -1.0f;           // 平移 X：-(R+L)/(R-L) = -1
    mat[13] = 1.0f;            // 平移 Y：-(T+B)/(T-B) = 1（因 T=0,B=H）
    mat[14] = 0.0f;
    mat[15] = 1.0f;
}
```
