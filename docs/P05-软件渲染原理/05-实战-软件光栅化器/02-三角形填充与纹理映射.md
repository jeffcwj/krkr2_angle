## 四、Step 3：三角形填充（扫描线法）

扫描线法核心：按 y 从上到下扫描，每行求左右交点区间 `[xL, xR]` 后填充。

### 4.1 平顶/平底三角形分解

通用三角形先按 y 排序：`v0.y <= v1.y <= v2.y`。

- 若 `v1.y == v2.y`：平底三角形。
- 若 `v0.y == v1.y`：平顶三角形。
- 否则在 `y = v1.y` 处切一刀，拆成一个平底 + 一个平顶。

扫描线示意图（ASCII）：

```text
           v0
          /  \
         /----\   <- y = y1 切分线
        /|    |\
       / |    | \
     v1--+----+--v1'
       \ |    | /
        \|    |/
         \----/
          v2

每条横线填充 [ceil(xL), floor(xR)]
```

代码示例 5（完整）：

```cpp
// 文件：main.cpp（片段）
struct Vec2f { float x, y; };

static void fillFlatBottom(FrameBuffer& fb, Vec2f v0, Vec2f v1, Vec2f v2, uint32_t color) {
    float invslope1 = (v1.x - v0.x) / (v1.y - v0.y);
    float invslope2 = (v2.x - v0.x) / (v2.y - v0.y);
    float curx1 = v0.x;
    float curx2 = v0.x;

    int yStart = std::max(0, (int)std::ceil(v0.y));
    int yEnd = std::min(fb.height - 1, (int)std::floor(v1.y));
    curx1 += (yStart - v0.y) * invslope1;
    curx2 += (yStart - v0.y) * invslope2;

    for (int y = yStart; y <= yEnd; ++y) {
        int xL = (int)std::ceil(std::min(curx1, curx2));
        int xR = (int)std::floor(std::max(curx1, curx2));
        xL = std::max(0, xL);
        xR = std::min(fb.width - 1, xR);
        for (int x = xL; x <= xR; ++x) fb.putPixelRaw(x, y, color);
        curx1 += invslope1;
        curx2 += invslope2;
    }
}

static void fillFlatTop(FrameBuffer& fb, Vec2f v0, Vec2f v1, Vec2f v2, uint32_t color) {
    float invslope1 = (v2.x - v0.x) / (v2.y - v0.y);
    float invslope2 = (v2.x - v1.x) / (v2.y - v1.y);
    int yStart = std::max(0, (int)std::ceil(v0.y));
    int yEnd = std::min(fb.height - 1, (int)std::floor(v2.y));
    float curx1 = v0.x + (yStart - v0.y) * invslope1;
    float curx2 = v1.x + (yStart - v1.y) * invslope2;

    for (int y = yStart; y <= yEnd; ++y) {
        int xL = (int)std::ceil(std::min(curx1, curx2));
        int xR = (int)std::floor(std::max(curx1, curx2));
        xL = std::max(0, xL);
        xR = std::min(fb.width - 1, xR);
        for (int x = xL; x <= xR; ++x) fb.putPixelRaw(x, y, color);
        curx1 += invslope1;
        curx2 += invslope2;
    }
}

static void fillTriangleScanline(FrameBuffer& fb, Vec2f a, Vec2f b, Vec2f c, uint32_t color) {
    if (a.y > b.y) std::swap(a, b);
    if (a.y > c.y) std::swap(a, c);
    if (b.y > c.y) std::swap(b, c);

    if ((int)std::round(b.y) == (int)std::round(c.y)) {
        fillFlatBottom(fb, a, b, c, color);
    } else if ((int)std::round(a.y) == (int)std::round(b.y)) {
        fillFlatTop(fb, a, b, c, color);
    } else {
        float t = (b.y - a.y) / (c.y - a.y);
        Vec2f d { a.x + t * (c.x - a.x), b.y };
        fillFlatBottom(fb, a, b, d, color);
        fillFlatTop(fb, b, d, c, color);
    }
}
```

## 五、Step 4：纹理映射（UV、透视校正、采样）

要点先说结论：

- 线性插值 `u,v` 在屏幕空间直接插会失真。
- 需要插值 `u/w、v/w、1/w`，最后再除回去，即透视校正。

代码示例 6（完整）：

```cpp
// 文件：main.cpp（片段）
struct Texture {
    int w = 0, h = 0;
    std::vector<uint32_t> data;

    uint32_t get(int x, int y) const {
        x = std::clamp(x, 0, w - 1);
        y = std::clamp(y, 0, h - 1);
        return data[size_t(y) * size_t(w) + size_t(x)];
    }

    uint32_t sampleNearest(float u, float v) const {
        int x = (int)std::round(u * (w - 1));
        int y = (int)std::round(v * (h - 1));
        return get(x, y);
    }

    uint32_t sampleBilinear(float u, float v) const {
        float fx = u * (w - 1);
        float fy = v * (h - 1);
        int x0 = (int)std::floor(fx), y0 = (int)std::floor(fy);
        int x1 = x0 + 1, y1 = y0 + 1;
        float tx = fx - x0, ty = fy - y0;

        auto lerp8 = [](int a, int b, float t) -> int { return (int)std::round(a + (b - a) * t); };
        auto c00 = get(x0, y0), c10 = get(x1, y0), c01 = get(x0, y1), c11 = get(x1, y1);

        int r0 = lerp8((c00 >> 16) & 255, (c10 >> 16) & 255, tx);
        int g0 = lerp8((c00 >> 8) & 255,  (c10 >> 8) & 255, tx);
        int b0 = lerp8(c00 & 255,         c10 & 255, tx);
        int a0 = lerp8((c00 >> 24) & 255, (c10 >> 24) & 255, tx);

        int r1 = lerp8((c01 >> 16) & 255, (c11 >> 16) & 255, tx);
        int g1 = lerp8((c01 >> 8) & 255,  (c11 >> 8) & 255, tx);
        int b1 = lerp8(c01 & 255,         c11 & 255, tx);
        int a1 = lerp8((c01 >> 24) & 255, (c11 >> 24) & 255, tx);

        int r = lerp8(r0, r1, ty), g = lerp8(g0, g1, ty), b = lerp8(b0, b1, ty), a = lerp8(a0, a1, ty);
        return (uint32_t(a) << 24) | (uint32_t(r) << 16) | (uint32_t(g) << 8) | uint32_t(b);
    }
};
```

片元阶段透视校正插值公式：

```text
给定顶点属性 u, v 与齐次 w：
先插值：u' = u / w, v' = v / w, q = 1 / w
得到像素后：u = u' / q, v = v' / q
```

## 六、Step 5：Alpha 混合集成（Porter-Duff Over）

实现 `Out = Src Over Dst`：

```text
Ao = As + Ad * (1 - As)
Co = (Cs*As + Cd*Ad*(1-As)) / Ao   （预乘格式可省一次除法）
```

我们这里使用常见非预乘输入，按 8bit 近似实现。

代码示例 7（完整）：

```cpp
// 文件：main.cpp（片段）
static inline uint32_t alphaOver(uint32_t dst, uint32_t src) {
    int sa = (src >> 24) & 255;
    int sr = (src >> 16) & 255;
    int sg = (src >> 8) & 255;
    int sb = src & 255;

    int da = (dst >> 24) & 255;
    int dr = (dst >> 16) & 255;
    int dg = (dst >> 8) & 255;
    int db = dst & 255;

    int invSa = 255 - sa;
    int outA = sa + ((da * invSa + 127) / 255);
    int outR = (sr * sa + dr * invSa + 127) / 255;
    int outG = (sg * sa + dg * invSa + 127) / 255;
    int outB = (sb * sa + db * invSa + 127) / 255;

    return (uint32_t(outA) << 24) | (uint32_t(outR) << 16) | (uint32_t(outG) << 8) | uint32_t(outB);
}
```

这一步和 KrKr2 `tvpgl.h`/`tvpgl.cpp` 的思想高度一致：

- `TVPAlphaBlend_*` 系列负责不同目标 Alpha 语义。
- `TVPOpacityOnOpacityTable`、`TVPNegativeMulTable` 通过查表加速。
- `TVPSaturatedAdd` 用位运算实现通道饱和加法，避免逐通道分支。

## 七、Step 6：SIMD 加速扫描线（SSE2）

扫描线填充时，热点往往是“连续像素混合”。可一次处理 4 像素。

代码示例 8（完整）：

```cpp
// 文件：main.cpp（片段）
#include <emmintrin.h>

static void blendSpanSSE2(uint32_t* dst, const uint32_t* src, int count) {
    int i = 0;
    const __m128i zero = _mm_setzero_si128();
    for (; i + 3 < count; i += 4) {
        __m128i s = _mm_loadu_si128(reinterpret_cast<const __m128i*>(src + i));
        __m128i d = _mm_loadu_si128(reinterpret_cast<const __m128i*>(dst + i));

        __m128i sLo = _mm_unpacklo_epi8(s, zero);
        __m128i sHi = _mm_unpackhi_epi8(s, zero);
        __m128i dLo = _mm_unpacklo_epi8(d, zero);
        __m128i dHi = _mm_unpackhi_epi8(d, zero);

        // 取源 alpha，复制到各通道
        __m128i aLo = _mm_shufflelo_epi16(sLo, _MM_SHUFFLE(3,3,3,3));
        aLo = _mm_shufflehi_epi16(aLo, _MM_SHUFFLE(3,3,3,3));
        __m128i aHi = _mm_shufflelo_epi16(sHi, _MM_SHUFFLE(3,3,3,3));
        aHi = _mm_shufflehi_epi16(aHi, _MM_SHUFFLE(3,3,3,3));

        __m128i invALo = _mm_sub_epi16(_mm_set1_epi16(255), aLo);
        __m128i invAHi = _mm_sub_epi16(_mm_set1_epi16(255), aHi);

        __m128i outLo = _mm_add_epi16(_mm_mullo_epi16(sLo, aLo), _mm_mullo_epi16(dLo, invALo));
        __m128i outHi = _mm_add_epi16(_mm_mullo_epi16(sHi, aHi), _mm_mullo_epi16(dHi, invAHi));

        outLo = _mm_srli_epi16(_mm_add_epi16(outLo, _mm_set1_epi16(127)), 8);
        outHi = _mm_srli_epi16(_mm_add_epi16(outHi, _mm_set1_epi16(127)), 8);

        __m128i out = _mm_packus_epi16(outLo, outHi);
        _mm_storeu_si128(reinterpret_cast<__m128i*>(dst + i), out);
    }
    for (; i < count; ++i) dst[i] = alphaOver(dst[i], src[i]);
}
```

说明：上面是教学版 SSE2，追求可读性；生产环境可继续把 Alpha 语义、预乘路径、边界对齐进一步细化。

## 八、Step 7：完整渲染管线（MVP 到屏幕）

代码示例 9（完整）：

```cpp
// 文件：main.cpp（片段）
struct Vec3 { float x, y, z; };
struct Vec4 { float x, y, z, w; };

struct Mat4 {
    float m[16]; // 列主序
    static Mat4 identity() {
        Mat4 r{};
        r.m[0] = r.m[5] = r.m[10] = r.m[15] = 1.0f;
        return r;
    }
};

static Vec4 mul(const Mat4& a, const Vec4& v) {
    Vec4 r{};
    r.x = a.m[0]*v.x + a.m[4]*v.y + a.m[8]*v.z + a.m[12]*v.w;
    r.y = a.m[1]*v.x + a.m[5]*v.y + a.m[9]*v.z + a.m[13]*v.w;
    r.z = a.m[2]*v.x + a.m[6]*v.y + a.m[10]*v.z + a.m[14]*v.w;
    r.w = a.m[3]*v.x + a.m[7]*v.y + a.m[11]*v.z + a.m[15]*v.w;
    return r;
}

static Vec3 ndcToScreen(const Vec3& n, int w, int h) {
    float sx = (n.x * 0.5f + 0.5f) * (float)(w - 1);
    float sy = (1.0f - (n.y * 0.5f + 0.5f)) * (float)(h - 1);
    return {sx, sy, n.z};
}
```

这里有三件事最容易被忽略：

1. 齐次裁剪应在 `-w <= x,y,z <= w` 空间进行。
2. 透视除法后才进入 NDC。
3. 屏幕映射时 y 轴通常要翻转（图像内存从上到下）。
