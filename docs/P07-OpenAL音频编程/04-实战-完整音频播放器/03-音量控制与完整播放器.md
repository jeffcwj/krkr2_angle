## OpenAL 音量体系

OpenAL 的音量控制分为三个层级：

```
最终音量 = Source Gain × Listener Gain × 距离衰减
```

| 层级 | API | 说明 |
|------|-----|------|
| Source 增益 | `alSourcef(src, AL_GAIN, g)` | 单个音源音量，0.0=静音 1.0=原始 |
| Listener 增益 | `alListenerf(AL_GAIN, g)` | 全局主音量，影响所有 Source |
| 距离衰减 | 自动计算 | 基于 Source/Listener 距离（见 Ch02-03）|

`AL_GAIN` 使用线性乘法（不是分贝），2.0 = 两倍增益（可能导致削波失真）。

## 示例1：基本音量控制

```cpp
#include <AL/al.h>
#include <AL/alc.h>
#include <cstdio>
#include <cstdint>
#include <vector>
#include <thread>
#include <chrono>
#include <cmath>

// WAV 头部和格式映射（同 Ch04-01）
struct WavHeader {
    char riff[4]; uint32_t fileSize; char wave[4]; char fmt[4];
    uint32_t fmtSize; uint16_t audioFormat; uint16_t channels;
    uint32_t sampleRate; uint32_t byteRate; uint16_t blockAlign;
    uint16_t bitsPerSample; char data[4]; uint32_t dataSize;
};

ALenum getALFormat(uint16_t ch, uint16_t bits) {
    if (ch == 1 && bits == 8)  return AL_FORMAT_MONO8;
    if (ch == 1 && bits == 16) return AL_FORMAT_MONO16;
    if (ch == 2 && bits == 8)  return AL_FORMAT_STEREO8;
    if (ch == 2 && bits == 16) return AL_FORMAT_STEREO16;
    return 0;
}

int main() {
    // 加载 WAV（简化：一次性加载到静态 Buffer）
    FILE* file = fopen("test.wav", "rb");
    if (!file) { fprintf(stderr, "无法打开 test.wav\n"); return 1; }
    WavHeader hdr;
    fread(&hdr, sizeof(hdr), 1, file);
    std::vector<char> pcm(hdr.dataSize);
    fread(pcm.data(), 1, hdr.dataSize, file);
    fclose(file);

    // OpenAL 初始化
    ALCdevice*  device  = alcOpenDevice(nullptr);
    ALCcontext* context = alcCreateContext(device, nullptr);
    alcMakeContextCurrent(context);

    ALuint buffer;
    alGenBuffers(1, &buffer);
    alBufferData(buffer, getALFormat(hdr.channels, hdr.bitsPerSample),
                 pcm.data(), (ALsizei)pcm.size(), hdr.sampleRate);

    ALuint source;
    alGenSources(1, &source);
    alSourcei(source, AL_BUFFER, buffer);

    // 设置初始音量为 50%
    alSourcef(source, AL_GAIN, 0.5f);
    printf("音量: 50%%\n");
    alSourcePlay(source);

    // 每秒增加 10% 音量
    for (int i = 6; i <= 10; ++i) {
        std::this_thread::sleep_for(std::chrono::seconds(1));
        float gain = i * 0.1f;
        alSourcef(source, AL_GAIN, gain);
        printf("音量: %d%%\n", i * 10);
    }

    // 等待播放结束
    ALint state = AL_PLAYING;
    while (state == AL_PLAYING) {
        std::this_thread::sleep_for(std::chrono::milliseconds(100));
        alGetSourcei(source, AL_SOURCE_STATE, &state);
    }

    alDeleteSources(1, &source);
    alDeleteBuffers(1, &buffer);
    alcMakeContextCurrent(nullptr);
    alcDestroyContext(context);
    alcCloseDevice(device);
    return 0;
}
```

## 淡入淡出（Fade In/Out）

人耳对音量的感知是对数关系：线性地将 `AL_GAIN` 从 0 增加到 1 会让人感觉"前面音量增长很快，后面几乎不变"。更自然的做法是使用**指数/对数曲线**：

```
线性淡入:  gain(t) = t / duration
对数淡入:  gain(t) = (exp(t/duration × k) - 1) / (exp(k) - 1)    k ≈ 5
平方淡入:  gain(t) = (t / duration)²
```

### 常见错误1：线性淡入听起来不自然

线性 Gain 变化在分贝域并非线性。例如 Gain 从 0.1 到 0.2 是 +6dB（响度翻倍），但从 0.8 到 0.9 只有 +1dB。解决方案：使用平方或指数曲线。

## 示例2：淡入淡出辅助函数

```cpp
#include <cmath>
#include <algorithm>

// 淡入：在 durationMs 毫秒内从 0 渐变到 targetGain
void fadeIn(ALuint source, float targetGain, int durationMs, int stepMs = 20) {
    int steps = durationMs / stepMs;
    for (int i = 1; i <= steps; ++i) {
        float t = (float)i / steps;          // 0..1
        float gain = targetGain * t * t;      // 平方曲线，更自然
        alSourcef(source, AL_GAIN, gain);
        std::this_thread::sleep_for(std::chrono::milliseconds(stepMs));
    }
    alSourcef(source, AL_GAIN, targetGain);   // 确保精确到达目标值
}

// 淡出：在 durationMs 毫秒内从当前音量渐变到 0
void fadeOut(ALuint source, int durationMs, int stepMs = 20) {
    float currentGain;
    alGetSourcef(source, AL_GAIN, &currentGain);
    int steps = durationMs / stepMs;
    for (int i = steps - 1; i >= 0; --i) {
        float t = (float)i / steps;
        float gain = currentGain * t * t;     // 平方曲线
        alSourcef(source, AL_GAIN, gain);
        std::this_thread::sleep_for(std::chrono::milliseconds(stepMs));
    }
    alSourcef(source, AL_GAIN, 0.0f);
}

// 交叉淡入淡出：src1 淡出，src2 同时淡入
void crossFade(ALuint src1, ALuint src2, float targetGain, int durationMs) {
    int stepMs = 20;
    int steps = durationMs / stepMs;
    float gain1;
    alGetSourcef(src1, AL_GAIN, &gain1);
    alSourcePlay(src2); // 启动新音源
    for (int i = 1; i <= steps; ++i) {
        float t = (float)i / steps;
        alSourcef(src1, AL_GAIN, gain1 * (1.0f - t) * (1.0f - t));
        alSourcef(src2, AL_GAIN, targetGain * t * t);
        std::this_thread::sleep_for(std::chrono::milliseconds(stepMs));
    }
    alSourceStop(src1);
    alSourcef(src2, AL_GAIN, targetGain);
}
```

## 声道平衡（Pan）

OpenAL 本身没有"pan"参数，但可以通过 `AL_POSITION` 模拟。将 Source 放在 Listener 左侧或右侧即可实现立体声平衡：

```cpp
// pan: -1.0 = 全左, 0.0 = 居中, 1.0 = 全右
void setPan(ALuint source, float pan) {
    // 将 Source 放在 X 轴上，Listener 默认在原点
    alSource3f(source, AL_POSITION, pan, 0.0f, 0.0f);
}
```

**注意：** 这种方法对 **Mono** 音源效果好，但对 **Stereo** 音源 OpenAL 默认不做空间化处理（立体声直接输出到左右声道）。如果需要对立体声音源做 pan，需要将其解码为 Mono 后再用 OpenAL 空间化，或在 PCM 层面手动调整左右声道增益。

### 常见错误2：Stereo 音源设置 Position 无效

许多开发者发现对 Stereo Source 调用 `alSource3f(AL_POSITION, ...)` 完全没有效果。这是 OpenAL 规范的设计：立体声数据已经包含空间信息，所以 OpenAL 不再对其做空间化。解决方案：使用 Mono 音源，或在 PCM 级别手动调整。

### KrKr2 中的 Pan 实现

KrKr2 在 `WaveMixer.cpp` 第 669-672 行使用了相同技巧：

```cpp
// WaveMixer.cpp (简化)
void tTVPSoundBufferAL::SetPan(float pan) {

```

KrKr2 的音量控制同样直接使用 `alSourcef(AL_GAIN)`（第 663-667 行）：

## 示例3：完整命令行 WAV 播放器

整合前面所有内容——WAV 解析、双线程流式播放、音量控制——构建一个完整的播放器：

```cpp
#include <AL/al.h>
#include <AL/alc.h>
#include <cstdio>
#include <cstdint>
#include <atomic>
#include <thread>
#include <vector>
#include <chrono>
#include <cstring>
#include <cmath>

// ===== WAV 解析 =====
struct WavHeader {
    char riff[4]; uint32_t fileSize; char wave[4]; char fmt[4];
    uint32_t fmtSize; uint16_t audioFormat; uint16_t channels;
    uint32_t sampleRate; uint32_t byteRate; uint16_t blockAlign;
    uint16_t bitsPerSample; char data[4]; uint32_t dataSize;
};
ALenum getALFormat(uint16_t ch, uint16_t bits) {
    if (ch == 1 && bits == 8)  return AL_FORMAT_MONO8;
    if (ch == 1 && bits == 16) return AL_FORMAT_MONO16;
    if (ch == 2 && bits == 8)  return AL_FORMAT_STEREO8;
    if (ch == 2 && bits == 16) return AL_FORMAT_STEREO16;
    return 0;
}

// ===== SPSC 环形缓冲区（精简版）=====
class SPSCRingBuffer {
    std::vector<char> buf_;
    size_t cap_;
    std::atomic<size_t> head_{0}, tail_{0};
public:
    explicit SPSCRingBuffer(size_t c) : buf_(c), cap_(c) {}
    size_t readAvailable() const {
        return (head_.load(std::memory_order_acquire)
              - tail_.load(std::memory_order_relaxed) + cap_) % cap_;
    }
    size_t write(const char* d, size_t n) {
        size_t avail = cap_ - 1 - ((head_.load(std::memory_order_relaxed)
                     - tail_.load(std::memory_order_acquire) + cap_) % cap_);
        if (n > avail) n = avail;
        size_t h = head_.load(std::memory_order_relaxed);
        for (size_t i = 0; i < n; ++i) buf_[(h + i) % cap_] = d[i];
        head_.store((h + n) % cap_, std::memory_order_release);
        return n;
    }
    size_t read(char* d, size_t n) {
        size_t avail = readAvailable();
        if (n > avail) n = avail;
        size_t t = tail_.load(std::memory_order_relaxed);
        for (size_t i = 0; i < n; ++i) d[i] = buf_[(t + i) % cap_];
        tail_.store((t + n) % cap_, std::memory_order_release);
        return n;
    }
};

// ===== 淡入淡出 =====
void fadeIn(ALuint src, float target, int ms, int step = 20) {
    int steps = ms / step;
    for (int i = 1; i <= steps; ++i) {
        float t = (float)i / steps;
        alSourcef(src, AL_GAIN, target * t * t);
        std::this_thread::sleep_for(std::chrono::milliseconds(step));
    }
    alSourcef(src, AL_GAIN, target);
}
void fadeOut(ALuint src, int ms, int step = 20) {
    float cur;
    alGetSourcef(src, AL_GAIN, &cur);
    int steps = ms / step;
    for (int i = steps - 1; i >= 0; --i) {
        float t = (float)i / steps;
        alSourcef(src, AL_GAIN, cur * t * t);
        std::this_thread::sleep_for(std::chrono::milliseconds(step));
    }
    alSourcef(src, AL_GAIN, 0.0f);
}

static constexpr int NUM_BUFFERS = 4;
static constexpr int BUF_SAMPLES = 4096;
static constexpr size_t RING_SIZE = 131072;

int main(int argc, char* argv[]) {
    if (argc < 2) {
        printf("用法: player <文件.wav> [音量0.0-1.0]\n");
        return 1;
    }
    float volume = (argc >= 3) ? (float)atof(argv[2]) : 1.0f;
    if (volume < 0.0f) volume = 0.0f;

    FILE* file = fopen(argv[1], "rb");
    if (!file) { fprintf(stderr, "无法打开: %s\n", argv[1]); return 1; }
    WavHeader hdr;
    fread(&hdr, sizeof(hdr), 1, file);
    ALenum format = getALFormat(hdr.channels, hdr.bitsPerSample);
    if (!format) { fprintf(stderr, "不支持的格式\n"); return 1; }
    int bytesPerFrame = hdr.channels * (hdr.bitsPerSample / 8);
    int bufBytes = BUF_SAMPLES * bytesPerFrame;
    printf("文件: %s | %dHz %dch %dbit | %.1fs\n",
           argv[1], hdr.sampleRate, hdr.channels,
           hdr.bitsPerSample, (float)hdr.dataSize / hdr.byteRate);

    // 解码线程
    SPSCRingBuffer ring(RING_SIZE);
    std::atomic<bool> decodeDone{false};
    std::thread decoder([&]() {
        std::vector<char> tmp(4096);
        while (true) {
            size_t n = fread(tmp.data(), 1, tmp.size(), file);
            if (n == 0) break;
            size_t w = 0;
            while (w < n) {
                size_t wrote = ring.write(tmp.data() + w, n - w);
                w += wrote;
                if (!wrote)
                    std::this_thread::sleep_for(std::chrono::milliseconds(1));
            }
        }
        decodeDone.store(true, std::memory_order_release);
        fclose(file);
    });

    // OpenAL
    ALCdevice*  dev = alcOpenDevice(nullptr);
    ALCcontext* ctx = alcCreateContext(dev, nullptr);
    alcMakeContextCurrent(ctx);
    ALuint source; alGenSources(1, &source);
    ALuint bufs[NUM_BUFFERS]; alGenBuffers(NUM_BUFFERS, bufs);
    alSourcef(source, AL_GAIN, 0.0f);

    // 预填充
    std::vector<char> pcm(bufBytes);
    int queued = 0;
    while (queued < NUM_BUFFERS) {
        if (ring.readAvailable() >= (size_t)bufBytes) {
            ring.read(pcm.data(), bufBytes);
            alBufferData(bufs[queued], format, pcm.data(), bufBytes, hdr.sampleRate);
            alSourceQueueBuffers(source, 1, &bufs[queued++]);
        } else if (decodeDone.load(std::memory_order_acquire)) {
            size_t r = ring.readAvailable();
            if (r > 0) {
                ring.read(pcm.data(), r);
                alBufferData(bufs[queued], format, pcm.data(), (ALsizei)r, hdr.sampleRate);
                alSourceQueueBuffers(source, 1, &bufs[queued++]);
            }
            break;
        } else { std::this_thread::sleep_for(std::chrono::milliseconds(1)); }
    }
    alSourcePlay(source);
    std::thread fader([&]() { fadeIn(source, volume, 500); });

    // 流式填充
    bool streamDone = false;
    while (true) {
        ALint processed = 0;
        alGetSourcei(source, AL_BUFFERS_PROCESSED, &processed);
        while (processed-- > 0 && !streamDone) {
            ALuint buf; alSourceUnqueueBuffers(source, 1, &buf);
            if (ring.readAvailable() >= (size_t)bufBytes) {
                ring.read(pcm.data(), bufBytes);
                alBufferData(buf, format, pcm.data(), bufBytes, hdr.sampleRate);
                alSourceQueueBuffers(source, 1, &buf);
            } else if (decodeDone.load(std::memory_order_acquire)) {
                size_t r = ring.readAvailable();
                if (r > 0) {
                    ring.read(pcm.data(), r);
                    alBufferData(buf, format, pcm.data(), (ALsizei)r, hdr.sampleRate);
                    alSourceQueueBuffers(source, 1, &buf);
                }
                streamDone = true;
            }
        }
        ALint state; alGetSourcei(source, AL_SOURCE_STATE, &state);
        if (streamDone && state != AL_PLAYING) break;
        ALint q = 0; alGetSourcei(source, AL_BUFFERS_QUEUED, &q);
        if (state != AL_PLAYING && q > 0 && !streamDone) alSourcePlay(source);
        std::this_thread::sleep_for(std::chrono::milliseconds(5));
    }

    fader.join();
    printf("播放结束\n");
    decoder.join();
    alDeleteSources(1, &source);
    alDeleteBuffers(NUM_BUFFERS, bufs);
    alcMakeContextCurrent(nullptr);
    alcDestroyContext(ctx);
    alcCloseDevice(dev);
    return 0;
}
```

### 常见错误3：淡入/淡出线程与填充循环的竞争

`fadeIn` 在独立线程调用 `alSourcef(AL_GAIN)`，主线程也调用 OpenAL 函数。OpenAL API 是线程安全的（同一 Context 内调用会被序列化），但如果淡入线程还在执行时又调用 `fadeOut`，两个线程交替设置 Gain 导致音量跳变。解决方案：使用 `std::atomic<float> targetGain`，由填充循环每帧平滑插值。

## 本节小结

- `alSourcef(AL_GAIN)` 控制音源音量，`alListenerf(AL_GAIN)` 控制全局主音量
- 淡入淡出用平方曲线比线性更自然（人耳是对数感知）
- Pan 通过 `alSource3f(AL_POSITION)` 实现，仅对 Mono 音源有效
- 完整播放器 = WAV解析 + 双线程流式 + 环形缓冲 + 音量控制 + 淡入

## 练习题与答案

### 题目1：`AL_GAIN = 2.0` 有什么风险？

<details>
<summary>查看答案</summary>

`AL_GAIN = 2.0` 将 PCM 振幅乘以 2（+6dB）。风险是**削波失真**：原始数据峰值接近满量程（16-bit 的 ±32767）时，乘 2 后超出范围被截断，产生明显噪音。安全做法：录音时留余量，或增益前检查峰值。

</details>

### 题目2：如何避免淡入/淡出线程冲突？

<details>
<summary>查看答案</summary>

用 `std::atomic<float> targetGain` 做唯一的音量指令通道，在填充循环中每帧平滑插值：

```cpp
std::atomic<float> targetGain{1.0f};
float currentGain = 0.0f;
// 填充循环每次迭代
float tg = targetGain.load(std::memory_order_relaxed);
currentGain += (tg - currentGain) * 0.05f; // 低通滤波
alSourcef(source, AL_GAIN, currentGain);
```

外部只设置 `targetGain`，避免多线程交替调用 `alSourcef`。

</details>

### 题目3：Stereo 音源不受 `AL_POSITION` 影响，如何变通？

<details>
<summary>查看答案</summary>

两种方案：
1. **PCM 级别调整**：读出左右声道采样值后手动乘以系数。`pan = -1.0` 时右声道 ×0，`pan = 1.0` 时左声道 ×0，中间线性插值
2. **拆为两个 Mono Source**：左声道和右声道各创建一个 Mono Buffer/Source，分别放在 `(-1,0,0)` 和 `(1,0,0)`

方案 1 更高效，KrKr2 对立体声也不使用空间化。

</details>
