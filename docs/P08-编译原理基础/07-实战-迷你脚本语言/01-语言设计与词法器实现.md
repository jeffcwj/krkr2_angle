## MiniScript 语言设计

### 设计目标

我们要设计的语言叫 **MiniScript**——一门极简的脚本语言：

| 特性 | 说明 | 对应字节码 |
|------|------|-----------|
| 数值 | 双精度浮点 `double` | `CONST`, `ADD`/`SUB`/`MUL`/`DIV`/`MOD` |
| 字符串 | 双引号 UTF-8 字符串 | `CONST`（常量池） |
| 变量 | `var x = 10;` 声明，`x = 20;` 赋值 | `MOV`/`CONST` |
| 算术 | `+ - * / %`，一元 `-x` | 对应算术指令 |
| 比较 | `== != < > <= >=` | `EQ`/`LT`/`LE` |
| 逻辑 | `and or not`（关键字） | `JF`（短路求值） |
| 控制流 | `if/else`、`while` | `JMP`/`JF` |
| 函数 | `fn add(a, b) { return a + b; }` | `CALL`/`RET` |
| 输出 | `print(expr);` | `PRINT` |
| 注释 | `//` 单行 | 词法器跳过 |

### 代码示例

```javascript
// 斐波那契数列
fn fib(n) {
    if (n <= 1) { return n; }
    return fib(n - 1) + fib(n - 2);
}

var i = 0;
while (i < 10) {
    print(fib(i));
    i = i + 1;
}
```

```javascript
// 阶乘
fn factorial(n) {
    if (n <= 1) { return 1; }
    return n * factorial(n - 1);
}
print(factorial(6));  // 720
```

### 完整 EBNF 文法

后续 Parser（下一节）严格遵循此文法：

```ebnf
program       = { declaration } ;
declaration   = varDecl | fnDecl | statement ;
varDecl       = "var" IDENTIFIER [ "=" expression ] ";" ;
fnDecl        = "fn" IDENTIFIER "(" [ paramList ] ")" block ;
paramList     = IDENTIFIER { "," IDENTIFIER } ;
statement     = exprStmt | printStmt | ifStmt | whileStmt | returnStmt | block ;
exprStmt      = expression ";" ;
printStmt     = "print" "(" expression ")" ";" ;
ifStmt        = "if" "(" expression ")" block [ "else" block ] ;
whileStmt     = "while" "(" expression ")" block ;
returnStmt    = "return" [ expression ] ";" ;
block         = "{" { declaration } "}" ;

expression    = assignment ;
assignment    = IDENTIFIER "=" assignment | logicOr ;
logicOr       = logicAnd { "or" logicAnd } ;
logicAnd      = equality  { "and" equality } ;
equality      = comparison { ( "==" | "!=" ) comparison } ;
comparison    = addition   { ( "<" | ">" | "<=" | ">=" ) addition } ;
addition      = multiplication { ( "+" | "-" ) multiplication } ;
multiplication = unary { ( "*" | "/" | "%" ) unary } ;
unary         = ( "-" | "not" ) unary | call ;
call          = primary { "(" [ argList ] ")" } ;
argList       = expression { "," expression } ;
primary       = NUMBER | STRING | IDENTIFIER | "true" | "false" | "(" expression ")" ;
```

### 运算符优先级（从低到高）

| 优先级 | 运算符 | 结合性 | 说明 |
|--------|--------|--------|------|
| 1 | `=` | 右结合 | 赋值 |
| 2 | `or` | 左结合 | 逻辑或 |
| 3 | `and` | 左结合 | 逻辑与 |
| 4 | `== !=` | 左结合 | 相等 |
| 5 | `< > <= >=` | 左结合 | 大小 |
| 6 | `+ -` | 左结合 | 加减 |
| 7 | `* / %` | 左结合 | 乘除模 |
| 8 | `- not` | 前缀 | 一元 |

### 与 TJS2 的对照

| 特性 | MiniScript | TJS2 |
|------|-----------|------|
| 变量声明 | `var x = 10;` | `var x = 10;` |
| 函数声明 | `fn foo(a) {}` | `function foo(a) {}` |
| 逻辑运算 | `and or not` | `&& \|\| !` |
| 类型系统 | `double` + `string` | Variant（void/int/real/string/object/octet） |
| 类/闭包 | 不支持 | 支持 |

## Token 类型定义

MiniScript 共有约 35 种 Token。完整定义如下：

```cpp
// mini_token.h —— Token 类型与结构
#pragma once
#include <string>
#include <iostream>

enum class TokenType {
    // 单字符
    LeftParen, RightParen, LeftBrace, RightBrace,
    Comma, Semicolon,
    Plus, Minus, Star, Slash, Percent,
    // 单/双字符
    Equal, EqualEqual,       // = ==
    BangEqual,               // !=
    Less, LessEqual,         // < <=
    Greater, GreaterEqual,   // > >=
    // 字面量
    Number, String, Identifier,
    // 关键字（12 个）
    Var, Fn, If, Else, While, Return, Print,
    True, False, And, Or, Not,
    // 特殊
    Eof, Error,
};

struct SourceLoc {
    int line   = 1;  // 行号（1-based）
    int column = 1;  // 列号（1-based）
};

struct Token {
    TokenType   type;
    std::string lexeme;  // 原始文本："3.14"、"var"、"hello"
    SourceLoc   loc;     // 源码位置
};
```

**关键设计：**
- `SourceLoc` 嵌入每个 Token，Parser 报错时能精确指向 "第几行第几列"
- `Error` Token：遇到非法字符不抛异常，产出 Error Token 后继续扫描——一次报告多个错误
- `lexeme` 存原始文本，数值转换延迟到 Compiler 阶段用 `std::stod()`

## Lexer 实现

### 类结构

```cpp
// mini_lexer.h
#pragma once
#include "mini_token.h"
#include <string>
#include <vector>
#include <unordered_map>

class Lexer {
    std::string source;
    int pos = 0, line = 1, col = 1;
    // 关键字表——O(1) 查询
    static const std::unordered_map<std::string, TokenType> keywords;
public:
    explicit Lexer(std::string src) : source(std::move(src)) {}
    std::vector<Token> scanAll();  // 批量扫描
    Token nextToken();             // 逐个产出
private:
    char peek() const;       // 查看当前字符
    char peekNext() const;   // 查看下一个字符
    char advance();          // 消费并推进
    bool isAtEnd() const;
    bool match(char expected);
    void skipWhitespace();
    Token makeToken(TokenType type, const std::string& lex, SourceLoc loc);
    Token scanNumber(SourceLoc loc);
    Token scanString(SourceLoc loc);
    Token scanIdentifier(SourceLoc loc);
};
```

### 辅助函数

```cpp
// mini_lexer.cpp
#include "mini_lexer.h"
#include <cctype>

const std::unordered_map<std::string, TokenType> Lexer::keywords = {
    {"var", TokenType::Var},     {"fn", TokenType::Fn},
    {"if", TokenType::If},       {"else", TokenType::Else},
    {"while", TokenType::While}, {"return", TokenType::Return},
    {"print", TokenType::Print}, {"true", TokenType::True},
    {"false", TokenType::False}, {"and", TokenType::And},
    {"or", TokenType::Or},       {"not", TokenType::Not},
};

char Lexer::peek() const {
    return isAtEnd() ? '\0' : source[pos];
}
char Lexer::peekNext() const {
    return (pos + 1 >= (int)source.size()) ? '\0' : source[pos + 1];
}
char Lexer::advance() {
    char ch = source[pos++];
    if (ch == '\n') { line++; col = 1; } else { col++; }
    return ch;
}
bool Lexer::isAtEnd() const { return pos >= (int)source.size(); }
bool Lexer::match(char expected) {
    if (isAtEnd() || source[pos] != expected) return false;
    advance();
    return true;
}
Token Lexer::makeToken(TokenType t, const std::string& lex, SourceLoc l) {
    return {t, lex, l};
}
```

### 跳过空白和注释

```cpp
void Lexer::skipWhitespace() {
    while (!isAtEnd()) {
        char ch = peek();
        if (ch == ' ' || ch == '\t' || ch == '\r' || ch == '\n') {
            advance();
        } else if (ch == '/' && peekNext() == '/') {
            // 单行注释：跳到行末
            while (!isAtEnd() && peek() != '\n') advance();
        } else {
            return;
        }
    }
}
```

### 扫描数字、字符串、标识符

```cpp
Token Lexer::scanNumber(SourceLoc startLoc) {
    int s = pos - 1;  // advance() 已消费第一个数字
    while (!isAtEnd() && std::isdigit(peek())) advance();
    // 小数点：必须后面跟数字（区分 42.toString）
    if (peek() == '.' && std::isdigit(peekNext())) {
        advance();  // 消费 '.'
        while (!isAtEnd() && std::isdigit(peek())) advance();
    }
    return makeToken(TokenType::Number, source.substr(s, pos - s), startLoc);
}

Token Lexer::scanString(SourceLoc startLoc) {
    std::string value;
    while (!isAtEnd() && peek() != '"') {
        if (peek() == '\n')
            return makeToken(TokenType::Error, "unterminated string", startLoc);
        if (peek() == '\\') {
            advance();  // 消费反斜杠
            if (isAtEnd())
                return makeToken(TokenType::Error, "bad escape", startLoc);
            char esc = advance();
            switch (esc) {
                case 'n':  value += '\n'; break;
                case 't':  value += '\t'; break;
                case '\\': value += '\\'; break;
                case '"':  value += '"';  break;
                default:
                    return makeToken(TokenType::Error,
                        std::string("unknown escape: \\") + esc, startLoc);
            }
        } else {
            value += advance();
        }
    }
    if (isAtEnd())
        return makeToken(TokenType::Error, "unterminated string", startLoc);
    advance();  // 消费闭合 "
    return makeToken(TokenType::String, value, startLoc);
}

Token Lexer::scanIdentifier(SourceLoc startLoc) {
    int s = pos - 1;
    while (!isAtEnd() && (std::isalnum(peek()) || peek() == '_'))
        advance();
    std::string lex = source.substr(s, pos - s);
    auto it = keywords.find(lex);
    // 关键字回查：先当标识符扫，再查哈希表
    return makeToken(it != keywords.end() ? it->second
                                          : TokenType::Identifier,
                     lex, startLoc);
}
```

**关键字回查**（maximal munch + 哈希查表）是手写 Lexer 的标准做法。TJS2 的 `tjsLex.cpp` 同样：先用 `TJS_iswalpha` 读完整单词，再用 `TJSReservedWordHash` 查关键字。

### 主扫描循环

```cpp
Token Lexer::nextToken() {
    skipWhitespace();
    if (isAtEnd()) return makeToken(TokenType::Eof, "", {line, col});

    SourceLoc startLoc = {line, col};  // 先记位置，再消费
    char ch = advance();

    if (std::isalpha(ch) || ch == '_') return scanIdentifier(startLoc);
    if (std::isdigit(ch))              return scanNumber(startLoc);
    if (ch == '"')                      return scanString(startLoc);

    switch (ch) {
        case '(': return makeToken(TokenType::LeftParen,  "(", startLoc);
        case ')': return makeToken(TokenType::RightParen, ")", startLoc);
        case '{': return makeToken(TokenType::LeftBrace,  "{", startLoc);
        case '}': return makeToken(TokenType::RightBrace, "}", startLoc);
        case ',': return makeToken(TokenType::Comma,      ",", startLoc);
        case ';': return makeToken(TokenType::Semicolon,  ";", startLoc);
        case '+': return makeToken(TokenType::Plus,  "+", startLoc);
        case '-': return makeToken(TokenType::Minus, "-", startLoc);
        case '*': return makeToken(TokenType::Star,  "*", startLoc);
        case '/': return makeToken(TokenType::Slash, "/", startLoc);
        case '%': return makeToken(TokenType::Percent, "%", startLoc);
        // 双字符运算符：先尝试两字符匹配
        case '=': return match('=')
            ? makeToken(TokenType::EqualEqual, "==", startLoc)
            : makeToken(TokenType::Equal, "=", startLoc);
        case '!': return match('=')
            ? makeToken(TokenType::BangEqual, "!=", startLoc)
            : makeToken(TokenType::Error, "use 'not' instead of '!'", startLoc);
        case '<': return match('=')
            ? makeToken(TokenType::LessEqual, "<=", startLoc)
            : makeToken(TokenType::Less, "<", startLoc);
        case '>': return match('=')
            ? makeToken(TokenType::GreaterEqual, ">=", startLoc)
            : makeToken(TokenType::Greater, ">", startLoc);
        default:
            return makeToken(TokenType::Error,
                std::string("unexpected '") + ch + "'", startLoc);
    }
}

std::vector<Token> Lexer::scanAll() {
    std::vector<Token> tokens;
    while (true) {
        Token tok = nextToken();
        tokens.push_back(tok);
        if (tok.type == TokenType::Eof) break;
    }
    return tokens;
}
```

### 常见错误与解决方案

**错误 1：`advance()` 后才记录 `startLoc`**

```cpp
// ❌ 位置指向第二个字符
char ch = advance();
SourceLoc loc = {line, col};  // 已经推进了！
// ✅ 先记录再消费
SourceLoc loc = {line, col};
char ch = advance();
```

**错误 2：双字符运算符顺序**

```cpp
// ❌ 永远匹配不到 ==
case '=': return makeToken(TokenType::Equal, "=", loc);
// ✅ 先尝试两字符
case '=': return match('=') ? ... : ...;
```

**错误 3：忘记处理转义序列**

```cpp
// ❌ 输入 "hello\n" 得到 7 字符 hello\n 而非 6 字符 hello+换行
while (peek() != '"') value += advance();  // 直接复制
// ✅ 检查 '\\'，按转义字符表替换
```

## 本节小结

- MiniScript 是一门极简脚本语言，覆盖变量、算术、比较、逻辑、控制流、函数——刚好对接前几章的 18 条字节码指令
- EBNF 文法是 Parser 的蓝图——每条产生式对应一个 Parser 函数（下一节实现）
- Lexer 的核心是 `nextToken()` 循环：跳空白 → 判字符类型 → 调子扫描器 → 产出 Token
- 关键字用"先扫后查"策略，转义序列用 switch-case 映射
- Error Token 机制让 Lexer 能一次报告多个词法错误

## 练习题与答案

### 题目 1：扩展 Lexer 支持多行注释 `/* ... */`

修改 `skipWhitespace()` 使其能跳过 `/* */` 注释，包括嵌套情况（可选）。

<details>
<summary>查看答案</summary>

```cpp
void Lexer::skipWhitespace() {
    while (!isAtEnd()) {
        char ch = peek();
        if (ch == ' ' || ch == '\t' || ch == '\r' || ch == '\n') {
            advance();
        } else if (ch == '/' && peekNext() == '/') {
            while (!isAtEnd() && peek() != '\n') advance();
        } else if (ch == '/' && peekNext() == '*') {
            advance(); advance(); // 消费 /*
            int depth = 1;        // 嵌套计数
            while (!isAtEnd() && depth > 0) {
                if (peek() == '/' && peekNext() == '*') {
                    advance(); advance();
                    depth++;     // 嵌套 +1
                } else if (peek() == '*' && peekNext() == '/') {
                    advance(); advance();
                    depth--;     // 嵌套 -1
                } else {
                    advance();
                }
            }
            // 如果 depth > 0，说明注释未闭合——可产出 Error Token
        } else {
            return;
        }
    }
}
```

</details>

### 题目 2：添加 `for` 关键字和十六进制数字

(a) 添加 `for` 关键字支持。(b) 扩展 `scanNumber()` 支持 `0xFF` 格式。

<details>
<summary>查看答案</summary>

**(a) for 关键字——只需两处改动：**

```cpp
// 1. TokenType 枚举添加 For
enum class TokenType { /* ..., */ For, /* ... */ };

// 2. 关键字表添加映射
{"for", TokenType::For},
```

Lexer 无需其他改动——`scanIdentifier()` 的关键字回查自动命中 `TokenType::For`。

**(b) 十六进制数字：**

```cpp
Token Lexer::scanNumber(SourceLoc startLoc) {
    int s = pos - 1;
    if (source[s] == '0' && !isAtEnd() && (peek()=='x' || peek()=='X')) {
        advance();  // 消费 'x'
        if (isAtEnd() || !std::isxdigit(peek()))
            return makeToken(TokenType::Error, "expected hex digit", startLoc);
        while (!isAtEnd() && std::isxdigit(peek())) advance();
        return makeToken(TokenType::Number, source.substr(s, pos-s), startLoc);
    }
    // 原有十进制逻辑...
    while (!isAtEnd() && std::isdigit(peek())) advance();
    if (peek()=='.' && std::isdigit(peekNext())) {
        advance();
        while (!isAtEnd() && std::isdigit(peek())) advance();
    }
    return makeToken(TokenType::Number, source.substr(s, pos-s), startLoc);
}
```

注意：十六进制转 double 需 `std::stoul(lex, nullptr, 16)` 再转，`std::stod("0xFF")` 不行。
